# Panel administrativo · Inicio

El **Panel administrativo** concentra la configuración, el control y el análisis del negocio en YURES.

## ¿Para qué sirve?

- Configurar parámetros clave del sistema (usuarios, permisos, impresión, estructura operativa).
- Supervisar indicadores de ventas, caja, inventario, compras y finanzas.
- Registrar y auditar operaciones administrativas con trazabilidad.

## URL de acceso

- <https://yures.puyu.pe/admin/>

## Qué encontrarás en esta sección

- Fichas del **Paquete 0 · Antes de iniciar** para preparar el acceso.
- Fichas operativas por módulos (caja, reportes, auditoría, configuración, catálogo, inventario, compras, finanzas y clientes).

> Recomendación: inicia por la ficha **0.0** antes de continuar con el login y las demás tareas.
