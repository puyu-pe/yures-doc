# 9.1 Cómo gestionar clientes en el Panel Administrativo – YURES

El módulo **Clientes y Fidelización → Lista y mantenimiento** permite registrar, editar, consultar y eliminar clientes del sistema.
Esta información es fundamental para:

* Emitir comprobantes correctamente.
* Llevar historial de consumo.
* Aplicar estrategias de fidelización.
* Gestionar reportes de ventas por cliente.
* Mantener una base de datos ordenada.

Una mala gestión en este módulo afecta directamente la facturación y los reportes comerciales.

---

# 📍 Paso 1: Ingresar al módulo

Desde el menú lateral:

**Clientes y Fidelización → Lista y mantenimiento**

Se mostrará la pantalla con:

* Listado de clientes registrados.
* Buscador.
* Paginación.
* Botón **+ Registrar**.
* Opciones de Editar y Eliminar.

---

# ➕ Paso 2: Registrar un nuevo cliente

Haga clic en:

**+ Registrar**

Se abrirá el formulario **Registro de Cliente**.

---

## 🧾 Campos obligatorios

### 1️⃣ Tipo de Documento

Seleccione el tipo de documento:

* DNI
* RUC
* Otros según configuración

⚠ Seleccione correctamente el tipo antes de ingresar el número.

---

### 2️⃣ Nº Documento

Ingrese el número correspondiente al tipo seleccionado.

Ejemplo:

* 8 dígitos para DNI.
* 11 dígitos para RUC.

Si el campo está vacío, el sistema mostrará validación obligatoria.

---

### 3️⃣ Nombre o Razón Social

Ingrese:

* Nombre completo si es persona natural.
* Razón social si es empresa.

Este campo es obligatorio.

---

## 🧾 Campos opcionales

### 4️⃣ Nº Celular

Permite registrar número de contacto del cliente.

Útil para:

* Promociones.
* Campañas.
* Seguimiento comercial.

---

### 5️⃣ Cumpleaños

Permite registrar fecha de cumpleaños.

Se utiliza para:

* Campañas de fidelización.
* Descuentos especiales.
* Recordatorios automáticos.

Formato: **dd/mm/aaaa**

---

### 6️⃣ Dirección

Ingrese dirección del cliente.

Importante para:

* Delivery.
* Referencias comerciales.
* Base de datos ordenada.

---

### 7️⃣ Referencia de la dirección

Campo complementario para detallar:

* Punto de referencia.
* Zona.
* Indicaciones adicionales.

---

### 8️⃣ Email

Permite registrar correo electrónico.

Útil para:

* Envío de comprobantes.
* Comunicaciones.
* Marketing.

---

### 9️⃣ Observación

Campo libre para registrar información adicional relevante.

Ejemplo:

* Cliente frecuente.
* Crédito autorizado.
* Condiciones especiales.

---

## 💾 Paso 3: Guardar cliente

Haga clic en:

**Guardar**

El cliente quedará registrado y disponible inmediatamente para:

* Ventas.
* Facturación.
* Reportes.
* Fidelización.

---

# 🔍 Paso 4: Buscar clientes

En la parte superior derecha encontrará el campo **Buscar**.

Puede buscar por:

* Número de documento.
* Nombre.
* Parte del nombre.
* Otros datos visibles.

Esto facilita localizar clientes rápidamente en bases de datos grandes.

---

# ✏ Paso 5: Editar cliente

En la columna **Opciones**, haga clic en el ícono de edición.

Puede modificar:

* Teléfono.
* Dirección.
* Email.
* Observaciones.
* Otros datos.

Luego presione **Guardar**.

⚠ Evite modificar número de documento si ya existen ventas asociadas.

---

# 🗑 Paso 6: Eliminar cliente

En la columna **Opciones**, haga clic en el ícono de eliminar.

El sistema mostrará mensaje de confirmación:

¿Realmente desea eliminar a este cliente?

Seleccione:

* **Aceptar** para eliminar.
* **Cancelar** para mantener el registro.

---

## ⚠ Recomendaciones antes de eliminar

* Verifique que el cliente no tenga historial de ventas.
* Revise si está asociado a comprobantes electrónicos.
* Consulte con administración si es un cliente recurrente.
* Evite eliminar clientes usados en reportes históricos.

Eliminar clientes con historial puede generar inconsistencias en reportes.

---

# 📊 Función dentro del sistema

El módulo de clientes permite:

* Mantener base de datos actualizada.
* Controlar clientes frecuentes.
* Realizar análisis de consumo.
* Implementar ranking por consumo.
* Aplicar estrategias de fidelización.
* Organizar facturación.

---

# 🧠 Buenas prácticas administrativas

* Registrar correctamente tipo y número de documento.
* No duplicar clientes.
* Estandarizar nombres.
* Completar datos de contacto.
* Revisar periódicamente la base.
* No eliminar registros históricos sin análisis.

---

# 🛑 Errores comunes

* Registrar el mismo cliente varias veces.
* Ingresar DNI en campo RUC o viceversa.
* No validar longitud del documento.
* Eliminar clientes con historial activo.
* No registrar información de contacto.

---

# 🎯 Resultado esperado

Al gestionar correctamente el módulo de clientes:

* Se mantiene orden en la base de datos.
* Se evita duplicidad.
* Se mejora control de ventas.
* Se facilita facturación.
* Se fortalece la estrategia de fidelización.
* Se mejora la trazabilidad comercial.

---

El módulo **Lista y mantenimiento de clientes** es una pieza clave dentro del Panel Administrativo de YURES para garantizar control, organización y eficiencia en la gestión comercial.
