# 3.3 Cómo investigar anulaciones y canjes cruzando operación y ventas en YURES

## Ruta
Caja, Turnos y Auditoría → Trazabilidad de operación / Comandas y Kitchen tracking; ruta relacionada: Ventas y Documentos → Búsqueda y gestión de comprobantes

## Objetivo
Aplicar el método operación → documento → corrección para explicar incidencias con evidencia.

## Precondiciones
- Haber detectado una incidencia de anulación o canje en un cierre o reporte.

## Pasos
1. Revisar la **Trazabilidad de operación** para identificar quién creó y modificó el pedido y en qué momentos.
2. Consultar **Comandas y Kitchen tracking** para verificar si el pedido fue preparado o despachado antes de la anulación/canje.
3. Ir a **Búsqueda y gestión de comprobantes** para localizar el documento asociado y revisar su estado y justificante.
4. Registrar la evidencia encontrada (usuarios implicados, tiempos, motivos) para sustentar la explicación de la incidencia.

## Resultado esperado
- Se reconstruye la secuencia de eventos que llevaron a la anulación o canje.
- Se documenta evidencia para respaldar la corrección o para capacitación del equipo.
