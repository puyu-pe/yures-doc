## 3.1 Cómo usar la Trazabilidad de Operación en Panel Administrativo

### Ruta

Caja, Turnos y Auditoría → Trazabilidad de operación

---

### Objetivo

Consultar el detalle completo de movimientos realizados en el sistema (ventas, productos, usuario, mesa, turno, etc.) dentro de un rango de fechas específico.

---

## 1. Configurar filtros de búsqueda

1. Ingresar a **Caja, Turnos y Auditoría**.
2. Hacer clic en **Trazabilidad de operación**.

Completar los siguientes campos:

* **Fecha inicio** → Seleccionar fecha inicial.
* **Fecha fin** → Seleccionar fecha final.
* **Piso** → Todos los pisos o uno específico.
* **Mesa** → Todas las mesas o una mesa específica.
* **Área** → Todas las áreas o una específica.
* **Mesero** → Todos los meseros o uno en particular.
* **Turnos** → Todos los turnos o turno específico.

3. Hacer clic en **Buscar**.

---

## 2. Revisar el reporte de movimientos

El sistema mostrará el **Reporte de Movimientos**, incluyendo:

* Documento
* Fecha de Caja
* Fecha y hora exacta
* Cantidad
* Ítem
* Categoría
* Precio
* Subtotal
* Piso
* Mesa
* Usuario
* Observación

Desplazarse verticalmente para revisar todos los registros.

---

## 3. Interpretación del reporte

### Permite verificar:

* Qué producto se vendió.
* En qué mesa y piso se registró.
* Qué usuario realizó la operación.
* Hora exacta del movimiento.
* Si hubo observaciones especiales (ej. modificaciones).

Es útil para auditoría interna y revisión de incidencias.

---

## 4. Función Imprimir

Después de realizar la búsqueda:

1. Hacer clic en **Imprimir**.
2. Se abrirá la vista previa de impresión.
3. Seleccionar impresora y confirmar.

---

## 5. Casos prácticos de uso

* Verificar reclamos de clientes sobre pedidos.
* Revisar operaciones de un mesero específico.
* Detectar inconsistencias en caja.
* Validar productos modificados o con observaciones.

---

## Buenas prácticas

* Usar rangos de fechas cortos para búsquedas más rápidas.
* Filtrar por mesero cuando se investiga una operación específica.
* Cruzar información con Cierre de Caja si se detecta diferencia.

---

## Advertencias

* Si el rango de fechas es amplio, la consulta puede tardar.
* Verificar que los filtros seleccionados correspondan al periodo real a investigar.

---

## Resultado esperado

El sistema mostrará el detalle completo de operaciones realizadas, permitiendo auditoría, validación de ventas y control interno del negocio.
