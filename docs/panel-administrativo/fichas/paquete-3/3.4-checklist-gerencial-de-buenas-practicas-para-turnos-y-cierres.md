# 3.4 Checklist gerencial de buenas prácticas para turnos y cierres en Panel Administrativo

## Ruta
Caja, Turnos y Auditoría → Gestión de Turnos + Cierres de caja

## Objetivo
Checklist de revisión operativa antes de validar cierres y turnos (sin enfoque contable).

## Precondiciones
- Tener la responsabilidad de revisar y aprobar turnos y cierres de caja.

## Pasos
1. Verificar que cada turno tenga registrado su horario de apertura y cierre, y que coincida con los reportes de terminal.
2. Revisar que se hayan imprimido o exportado los reportes de medios de pago y comprobantes asociados al periodo.
3. Confirmar que no existan ventas sin cerrar ni comandas pendientes antes de aprobar el cierre.
4. Revisar justificativos de anulaciones, canjes y notas de crédito para asegurar que cuentan con la evidencia necesaria.
5. Registrar observaciones y aprobar el cierre si toda la información coincide.

## Resultado esperado
- El gerente aprueba cierres y turnos con confianza, reduciendo descuadres y cuestionamientos posteriores.
- Se documentan observaciones para seguimiento y mejora continua.
