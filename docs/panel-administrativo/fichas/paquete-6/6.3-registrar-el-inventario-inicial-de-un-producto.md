# 6.3 Cómo registrar el inventario inicial de un producto en el Panel Administrativo – YURES

El registro de **inventario inicial** es el proceso mediante el cual se ingresan al sistema las cantidades reales con las que comienza a operar un almacén. Este paso es fundamental cuando:

* Se implementa YURES por primera vez.
* Se crea un nuevo almacén.
* Se incorporan productos nuevos.
* Se regulariza inventario tras una auditoría física.

Si no se registra correctamente el inventario inicial, el sistema puede mostrar:

* Stock negativo.
* Cantidades incorrectas.
* Inconsistencias en el Kardex.
* Errores en costos o reportes.

---

## 📍 Ruta de acceso

Menú lateral:

**Inventario, Recetas y Costeo → Stock por almacén**

Asegúrese de ubicarse en la pestaña correcta del almacén donde desea registrar el inventario.

---

# 1️⃣ Verificar el almacén correcto

En la parte superior encontrará las pestañas:

* Almacén
* Almacén Ventas
* Almacén General

Seleccione el almacén donde físicamente se encuentra el producto.

⚠ Es importante registrar el inventario inicial en el almacén correcto, ya que cada almacén maneja stock independiente.

---

# 2️⃣ Presionar el botón “+ Registrar”

En la parte superior derecha encontrará el botón azul:

**+ Registrar**

Este botón permite ingresar un producto con su cantidad inicial en el almacén seleccionado.

También puede hacerlo desde el botón verde “+” en la columna Opciones si el producto ya figura en la lista.

---

# 3️⃣ Completar el formulario “Productos por almacén”

Se abrirá una ventana emergente con los siguientes campos:

* **Item (Producto)**
* **Almacén**
* **Cantidad**
* **Observación**

---

## 📝 Detalle de cada campo

### ✔ Item (Producto)

Seleccione el producto desde la lista desplegable.

Puede:

* Escribir el nombre para buscarlo.
* Desplazarse por la lista.
* Seleccionar el producto exacto.

Ejemplo:
AGUA MINERAL (CON GAS) 500 ml
ACEITE VEGETAL Lt

---

### ✔ Almacén

El sistema mostrará el almacén activo.

Verifique que sea el correcto antes de continuar.

---

### ✔ Cantidad

Ingrese la cantidad real disponible físicamente.

Ejemplo:

* 10 (si son 10 litros)
* 55 (si son 55 kilos)
* 75 (si son 75 unidades)

Debe ingresar solo números, sin texto adicional.

---

### ✔ Observación

En este campo se recomienda escribir:

* “Inventario inicial”
* “Carga inicial”
* “Inicio de sistema”
* “Regularización física”

Este campo es importante para trazabilidad futura en auditorías.

---

# 4️⃣ Guardar el registro

Una vez completados los datos:

Presione el botón **Guardar**.

El sistema:

* Registrará el movimiento.
* Actualizará el stock automáticamente.
* Mostrará la cantidad en la tabla principal.

---

# 5️⃣ Verificar el resultado

Después de guardar, el producto aparecerá en la lista con la cantidad registrada.

Ejemplo:

| Producto          | Unidad | Cantidad |
| ----------------- | ------ | -------- |
| ACEITE VEGETAL Lt | Lt     | 10.00    |
| AZUCAR BLANCA Kg  | Kg     | 55.00    |

Esto confirma que el inventario inicial fue cargado correctamente.

---

# 6️⃣ Qué sucede internamente

Al registrar inventario inicial:

* Se genera un movimiento de ingreso.
* Se actualiza el stock del almacén.
* Se registra en el Kardex.
* Se afecta la trazabilidad histórica.

Por eso es importante no duplicar registros.

---

# 7️⃣ Errores comunes que debe evitar

❌ Registrar inventario inicial más de una vez.
❌ Ingresar cantidades sin conteo físico previo.
❌ Registrar en el almacén incorrecto.
❌ No colocar observación.
❌ Registrar inventario inicial cuando ya existen movimientos activos.

---

# 8️⃣ Buenas prácticas recomendadas

✔ Realizar conteo físico antes de registrar.
✔ Registrar todos los productos el mismo día.
✔ Documentar el motivo en observación.
✔ Revisar inmediatamente la tabla luego de guardar.
✔ Validar en Kardex si es necesario.

---

# 9️⃣ Cuándo usar esta función

Use el registro de inventario inicial cuando:

* El sistema inicia operación.
* Se crea un nuevo almacén.
* Se incorporan nuevos productos al catálogo.
* Se hace corte contable y se reinicia inventario.

---

# 🎯 Resultado esperado

Al finalizar el proceso correctamente:

* El sistema reflejará el stock real.
* No habrá productos en negativo.
* Los reportes serán coherentes.
* La operación iniciará con base confiable.
* El Kardex mostrará trazabilidad desde el inicio.

---

El registro de inventario inicial no es un ajuste común; es la base del control de inventarios. Debe realizarse con precisión y solo cuando corresponda para mantener consistencia operativa y financiera en YURES.
