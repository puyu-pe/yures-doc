# 6.4 Cómo realizar ajustes de inventario (ingresos y salidas) en el Panel Administrativo – YURES

El ajuste de inventario permite **corregir manualmente las cantidades** de un producto dentro de un almacén específico. Se utiliza cuando existe una diferencia entre el stock físico real y el stock registrado en el sistema.

Este procedimiento puede generar:

* **Ingresos** (aumenta el stock)
* **Salidas** (disminuye el stock)

Es una herramienta de control, no debe usarse como reemplazo de compras o ventas.

---

## 📍 Ruta de acceso

Menú lateral:

**Inventario, Recetas y Costeo → Stock por almacén**

---

# 1️⃣ Ubicar el almacén correcto

En la parte superior encontrará las pestañas:

* Almacén
* Almacén Ventas
* Almacén General

Seleccione el almacén donde desea realizar el ajuste.

⚠ Cada almacén maneja su propio stock. Verifique antes de continuar.

---

# 2️⃣ Identificar el producto a ajustar

En la tabla encontrará:

* Categoría
* Producto
* Unidad
* Cantidad actual
* Opciones

En la columna **Opciones** verá un botón verde con el símbolo **+**.

Este botón permite realizar el ajuste directamente sobre ese producto.

---

# 3️⃣ Abrir el formulario de ajuste

Haga clic en el botón verde **+** del producto correspondiente.

Se abrirá la ventana:

### “Registro de ajuste”

Contiene los siguientes campos:

* Almacén
* Producto
* Tipo
* Cantidad
* Observación
* Botones: Cerrar / Guardar

---

# 4️⃣ Completar el formulario correctamente

## ✔ Almacén

Se mostrará automáticamente el almacén activo.

No puede modificarlo desde este formulario.

---

## ✔ Producto

Se mostrará el producto seleccionado.

Verifique que sea el correcto antes de continuar.

---

## ✔ Tipo

Debe seleccionar el tipo de ajuste:

* **Ingreso** → cuando necesita aumentar el stock.
* **Salida** → cuando necesita disminuir el stock.

Ejemplos:

| Situación                      | Tipo             |
| ------------------------------ | ---------------- |
| Conteo físico mayor al sistema | Ingreso          |
| Producto dañado o perdido      | Salida           |
| Error previo de registro       | Depende del caso |

---

## ✔ Cantidad

Ingrese la cantidad exacta a ajustar.

⚠ No ingrese la cantidad total final, sino la cantidad que va a sumar o restar.

Ejemplo:

Si el sistema muestra 45 Kg y el conteo físico real es 50 Kg:

* Tipo: Ingreso
* Cantidad: 5

Si el sistema muestra 45 Kg y el conteo real es 40 Kg:

* Tipo: Salida
* Cantidad: 5

---

## ✔ Observación

Este campo es obligatorio desde el punto de vista de control interno.

Ejemplos recomendados:

* Ajuste por conteo físico
* Merma por deterioro
* Producto vencido
* Corrección de error
* Regularización inventario

Nunca deje este campo vacío en auditorías formales.

---

# 5️⃣ Guardar el ajuste

Después de completar los campos:

Presione **Guardar**.

El sistema:

* Actualiza la cantidad inmediatamente.
* Registra el movimiento.
* Refleja el cambio en la tabla principal.
* Genera trazabilidad en Kardex.

---

# 6️⃣ Verificar el resultado

Después de guardar:

* Revise la columna **Cantidad**.
* Confirme que el nuevo saldo sea correcto.
* Si es necesario, valide en el módulo **Kardex**.

---

# 7️⃣ Cuándo realizar un ajuste

Use esta función en los siguientes casos:

✔ Diferencia entre conteo físico y sistema.
✔ Pérdida o merma.
✔ Producto vencido.
✔ Corrección de error humano.
✔ Regularización contable.
✔ Inicio de inventario después de auditoría.

---

# 8️⃣ Cuándo NO usar ajustes

No debe utilizar ajustes para:

❌ Registrar compras normales.
❌ Registrar producción.
❌ Simular ventas.
❌ Mover productos entre almacenes.

Para esos casos existen módulos específicos:

* Movimientos de inventario
* Transferencias internas
* Producción
* Ventas

El uso indebido de ajustes distorsiona reportes financieros.

---

# 9️⃣ Buenas prácticas

✔ Realizar conteo físico antes de ajustar.
✔ Documentar claramente la observación.
✔ No realizar ajustes masivos sin respaldo.
✔ Revisar el Kardex periódicamente.
✔ Evitar ajustes frecuentes sin causa clara.

---

# 🔟 Impacto en el sistema

Cada ajuste:

* Modifica el stock del almacén.
* Genera movimiento histórico.
* Afecta reportes de inventario.
* Puede impactar análisis de costos.

Por eso debe utilizarse con criterio administrativo.

---

# 🎯 Resultado esperado

Al realizar correctamente los ajustes de inventario:

* El sistema reflejará la realidad física.
* Se eliminarán saldos negativos.
* Se mantendrá coherencia operativa.
* Los reportes serán confiables.
* La gestión administrativa será precisa.

---

El módulo de ajustes es una herramienta de control correctivo. Debe utilizarse con disciplina, trazabilidad y respaldo físico para mantener integridad en el sistema de inventarios de YURES.
