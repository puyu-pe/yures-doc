# 6.1 Cómo gestionar almacenes en el Panel Administrativo – YURES

La **Gestión de Almacenes** es una sección fundamental dentro del módulo de inventario. Desde aquí se crean, editan y organizan los distintos almacenes que el sistema utilizará para controlar existencias, movimientos internos, producción y ventas.

Una correcta configuración de almacenes permite:

* Separar inventario físico por áreas (cocina, barra, producción, ventas).
* Controlar entradas y salidas de stock.
* Realizar transferencias internas.
* Generar reportes confiables.
* Evitar descuadres en inventario.

---

## 📍 Ruta de acceso

Menú lateral:

**Inventario, Recetas y Costeo → Gestión de almacenes**

Al ingresar, el sistema mostrará:

* Listado de almacenes existentes.
* Botón **+ Registrar**.
* Campo de búsqueda.
* Opciones para editar o eliminar.

---

# 1️⃣ Vista general del listado de almacenes

En la pantalla principal se visualizan las siguientes columnas:

* **Almacén** → Nombre del almacén.
* **Tipo** → Clasificación del almacén.
* **Opciones** → Editar o eliminar.

Ejemplo de almacenes comunes:

* Almacén General
* Almacén Ventas
* Cocina
* Producción

Cada uno cumple una función específica dentro del flujo operativo.

---

# 2️⃣ Registrar un nuevo almacén

Para crear un almacén:

1. Presione el botón **+ Registrar**.
2. Se abrirá una ventana emergente.
3. Complete los campos solicitados.
4. Presione **Guardar**.

---

## 🔹 Campos del formulario

### 📌 Descripción

Ingrese el nombre del almacén.

Ejemplos:

* Cocina Principal
* Barra
* Producción Central
* Almacén Seco
* Almacén Refrigerado

Se recomienda utilizar nombres claros y coherentes con la operación real del negocio.

---

### 📌 Tipo

Debe seleccionar el tipo de almacén desde el desplegable:

Opciones disponibles:

* Producción
* Venta
* Venta General
* Almacén

Cada tipo tiene una función específica.

---

## 🔎 ¿Qué significa cada tipo?

### 🔹 Producción

Se utiliza para procesos internos donde se transforman insumos.

Ejemplo:

* Producción de masas
* Preparación base
* Fabricación interna

Este tipo de almacén es clave cuando se trabaja con recetas y producción interna.

---

### 🔹 Venta

Almacén destinado directamente a la venta.

Ejemplo:

* Almacén de mostrador
* Punto de despacho
* Caja

Aquí se descuentan productos vendidos.

---

### 🔹 Venta General

Se utiliza como almacén principal vinculado al proceso general de ventas.

Suele configurarse cuando se desea centralizar los descuentos automáticos del inventario por venta.

---

### 🔹 Almacén

Es un almacén general sin una clasificación específica.

Ejemplo:

* Almacén seco
* Almacén de insumos
* Depósito principal

---

# 3️⃣ Guardar el almacén

Una vez completados los campos:

1. Presione **Guardar**.
2. El sistema registrará el almacén.
3. Aparecerá automáticamente en el listado.

Si desea cancelar, presione **Cerrar**.

---

# 4️⃣ Editar un almacén

Para modificar un almacén existente:

1. Presione el ícono de **lápiz** en la columna Opciones.
2. Se abrirá la ventana con los datos actuales.
3. Realice los cambios necesarios.
4. Presione **Guardar**.

Se puede modificar:

* Nombre
* Tipo

---

⚠ Recomendación importante:

Evite cambiar el tipo de almacén si ya tiene movimientos registrados, ya que podría afectar reportes o procesos internos.

---

# 5️⃣ Eliminar un almacén

Para eliminar:

1. Presione el ícono de **basurero**.
2. El sistema mostrará una confirmación.
3. Presione **Aceptar** si está seguro.

---

⚠ Advertencias importantes:

* No elimine almacenes que ya tengan movimientos de inventario.
* Si el almacén está vinculado a recetas o producción, podría generar inconsistencias.
* Lo recomendable es descontinuar su uso antes de eliminarlo.

---

# 6️⃣ Uso del buscador

En la parte superior derecha encontrará el campo **Buscar**.

Permite:

* Localizar rápidamente un almacén.
* Filtrar por nombre.
* Verificar duplicados.

Es útil cuando el negocio maneja múltiples almacenes.

---

# 7️⃣ Buenas prácticas en la gestión de almacenes

✔ Crear solo los almacenes necesarios.
✔ No duplicar nombres.
✔ Separar almacenes físicos reales.
✔ Definir claramente cuál será el almacén de venta.
✔ Evitar eliminar almacenes con historial.
✔ Revisar configuración antes de iniciar operaciones.

---

# 8️⃣ Cómo influye en el control de inventario

La correcta gestión de almacenes impacta directamente en:

* Movimientos de inventario.
* Transferencias internas.
* Producción.
* Kardex.
* Reportes de stock por almacén.
* Costeo real de recetas.

Si los almacenes están mal configurados, el inventario perderá trazabilidad.

---

# 🎯 Resultado esperado

Al gestionar correctamente los almacenes en YURES:

* Tendrá control real del flujo físico de productos.
* Podrá separar producción de ventas.
* Mejorará la precisión del stock.
* Optimizará reportes financieros.
* Reducirá errores operativos.

---

La Gestión de Almacenes no es solo una configuración inicial; es la base estructural sobre la cual funciona todo el módulo de inventario del sistema. Una correcta implementación garantiza orden, control y eficiencia operativa en el negocio.
