# 6.7 Cómo revisar el kardex en Panel Administrativo

## Ruta
Inventario, Recetas y Costeo → Kardex

## Objetivo
Consultar historial y valorización para auditoría y lectura gerencial.

## Precondiciones
- Tener movimientos de inventario generados para el producto a consultar.

## Pasos
1. Acceder al módulo **Kardex**.
2. Seleccionar el producto y el periodo para el cual se requiere el historial de movimientos y valorizaciones.
3. Revisar el listado de entradas y salidas, con fechas, cantidades, costos unitarios y saldo valorizado.
4. Exportar o imprimir el kardex si es necesario para documentación contable o auditoría.

## Resultado esperado
- El sistema presenta el historial completo de movimientos y valorización del producto seleccionado.
- Se facilita el control de inventarios y la conciliación contable.
