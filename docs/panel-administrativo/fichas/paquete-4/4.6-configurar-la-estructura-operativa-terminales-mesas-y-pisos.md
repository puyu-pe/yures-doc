## 4.6 Cómo configurar la estructura operativa (terminales, mesas y pisos) en Panel Administrativo

---

### Ruta

Configuración del sistema → Estructura operativa → Terminales

---

## Objetivo

Registrar, editar y eliminar terminales que operan en el sistema (caja, toma, assistant), y asignar categorías cuando corresponda.

---

# 1️⃣ Registrar una terminal

### Precondiciones

Tener acceso a Configuración del sistema.

### Pasos

1. Ingresar a Configuración del sistema → Estructura operativa → Terminales.
2. Hacer clic en + Registrar.
3. Completar el formulario:

   * Tipo
   * Nombre
   * Nombre del Sistema
   * IP
   * Nombre Impresora
   * Tipo de Maquina
   * Api Key
4. Hacer clic en Guardar.

### Resultado esperado

La terminal se muestra en la tabla de TERMINALES con sus datos registrados.

---

# 2️⃣ Editar una terminal

### Pasos

1. Ubicar la terminal en la tabla.
2. Hacer clic en el ícono Editar.
3. Modificar los campos necesarios.
4. Hacer clic en Guardar cambios.

### Resultado esperado

La tabla actualiza la información modificada.

---

# 3️⃣ Eliminar una terminal

### Pasos

1. Ubicar la terminal en la tabla.
2. Hacer clic en el ícono Eliminar.
3. Confirmar en la ventana ¿Realmente desea eliminar este terminal? con Aceptar.

### Resultado esperado

La terminal desaparece de la tabla de TERMINALES.

---

# 4️⃣ Asignar categorías a una terminal

### Pasos

1. Ubicar la terminal en la tabla.
2. Hacer clic en el ícono de Categorías.
3. Seleccionar las categorías disponibles (por ejemplo: BEBIDAS, PIZZAS, POLLOS A LA BRAZA, etc.).
4. Cerrar la ventana.

### Resultado esperado

El sistema muestra el mensaje Categoría seleccionada correctamente.

---

## Nota operativa

Las terminales tipo toma o tablet pueden configurarse para mostrar solo determinadas categorías en el POS.

---

## Advertencia

Eliminar una terminal en uso puede afectar la operación en el POS si el dispositivo sigue activo.

---

## Recomendación

Registrar una terminal por cada dispositivo físico (PC, Tablet, PC Touch) para mantener trazabilidad.

---

## Relación con otros módulos

La configuración de Terminales impacta directamente en el POS, ya que define:

* Qué dispositivo puede vender.
* Qué impresora utilizar.
* Qué categorías visualizar.

Si la terminal no está registrada correctamente, el POS no operará de manera adecuada.
