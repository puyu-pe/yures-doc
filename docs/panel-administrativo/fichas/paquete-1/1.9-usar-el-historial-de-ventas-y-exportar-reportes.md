# 1.9 Cómo usar el historial de ventas y exportar reportes en Panel Administrativo

### Ruta

Ventas y Documentos → Historial de ventas

---

### Objetivo

Generar reportes de ventas por rango de fechas, terminal o turno, y exportarlos.

---

## Generar reporte

1. Ingresar a **Ventas y Documentos → Historial de ventas**.
2. En el campo **Tipo**, seleccionar el formato de reporte:

   * Por detalle
   * Por medio de pago
   * Otros disponibles
3. En **Terminal**, seleccionar la caja correspondiente.
4. En **Turno**, seleccionar el turno o dejar **- Todo -**.
5. En **Fecha inicio**, seleccionar la fecha inicial.
6. En **Fecha fin**, seleccionar la fecha final.
7. Hacer clic en **Generar reporte**.

El sistema mostrará la **LISTA DE VENTA**.

---

## Revisar información del reporte

Verificar las columnas visibles:

* DOCUMENTO
* SERIE-CORRELATIVO
* CLIENTE
* MEDIO(S) DE PAGO
* ESTADO
* TOTAL
* USUARIO
* HORA

Desplazarse verticalmente para revisar todos los registros.

---

## Imprimir reporte

1. Hacer clic en **IMPRIMIR**.

El sistema generará la vista lista para impresión.

---

## Exportar reporte a Excel

1. Hacer clic en **EXPORTAR EXCEL**.

El sistema descargará el archivo en formato Excel.

---

### Nota operativa

* El reporte se genera según los filtros aplicados.
* Si no se selecciona Terminal o Turno, se mostrarán todos los registros dentro del rango de fechas.

---

### Advertencia

* Verificar correctamente el rango de fechas antes de generar el reporte.
* Si no existen ventas en el periodo seleccionado, el reporte aparecerá vacío.

---

### Recomendación

* Generar reportes diarios por turno para control de caja.
* Exportar a Excel para conciliaciones contables o revisión externa.

---

### Relación con otros módulos

* Caja, Turnos y Auditoría → Permite filtrar ventas por turno de trabajo.
* PANEL OPERATIVO → Todas las ventas registradas en PANEL OPERATIVO se reflejan en este historial.
* Ventas y Documentos → Los comprobantes emitidos aparecen en este reporte.

---

### Resultado esperado

El sistema muestra el reporte de ventas filtrado y permite imprimirlo o exportarlo en Excel.
