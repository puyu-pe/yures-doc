# 1.4 Cómo revisar turnos de auditoría en Panel Administrativo

## Ruta
Caja, Turnos y Auditoría → Gestión de Turnos

## Objetivo
Revisar operador, horarios, terminal y eventos relevantes antes de validar un cierre.

## Precondiciones
- Contar con permisos de auditoría para acceder a la gestión de turnos.
- Tener turnos registrados en el sistema.

## Pasos
1. Ingresar al módulo **Caja, Turnos y Auditoría** y seleccionar la opción **Gestión de Turnos**.
2. Utilizar los filtros disponibles para elegir la terminal y el rango de fechas del turno que desea revisar.
3. Hacer clic en **Buscar** para listar los turnos registrados.
4. Revisar la información del operador, horario de inicio y cierre, terminal asociada y eventos (aperturas, cierres, incidencias) en la tabla.
5. Si necesita más detalles, seleccionar el turno para acceder a la vista ampliada o historial de acciones.

## Resultado esperado
- Se obtiene una lista de turnos filtrados con detalles completos de cada operador y terminal.
- El auditor puede validar horarios y eventos antes de aprobar el cierre de caja.
