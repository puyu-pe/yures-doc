# 1.11 Cómo usar el historial de ventas y exportar reportes en Panel Administrativo

## Ruta
Ventas y Documentos → Historial de ventas

## Objetivo
Aplicar filtros clave y exportar (PDF/Excel) para control y contabilidad.

## Precondiciones
- Disponer de ventas registradas en el periodo a consultar.

## Pasos
1. Acceder al módulo **Historial de ventas** dentro de **Ventas y Documentos**.
2. Seleccionar el rango de fechas, tipo de documento o estado que se desea consultar utilizando los filtros disponibles.
3. Hacer clic en **Buscar** para visualizar las ventas correspondientes al criterio seleccionado.
4. Revisar el listado obtenido y utilizar la paginación si hay muchos registros.
5. Exportar el reporte a PDF o Excel mediante los botones de **Exportar** según el formato deseado.

## Resultado esperado
- Se muestra una tabla con todas las ventas filtradas y su detalle principal.
- El usuario descarga el archivo en el formato seleccionado para respaldo contable.
