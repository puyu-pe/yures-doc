# 1.9 Cómo canjear un ticket a boleta o factura en Panel Administrativo

## Ruta
Ventas y Documentos → Búsqueda y gestión de comprobantes → Seleccionar… → Canjear ticket

## Objetivo
Canjear validando cliente, datos y monto para evitar incidencias.

## Precondiciones
- Tener el ticket o comprobante que se desea canjear.
- Disponer de datos completos del cliente (nombre, documento de identidad) para emisión de boleta o factura.

## Pasos
1. Buscar el ticket en la lista de comprobantes y seleccionar **Canjear ticket** en su fila.
2. Elegir el tipo de comprobante de destino (boleta o factura) y verificar el importe.
3. Ingresar o actualizar los datos del cliente en la ventana de canje.
4. Confirmar el canje; el sistema emitirá la nueva boleta o factura y actualizará el estado del ticket.

## Resultado esperado
- Se genera un nuevo comprobante con la información fiscal correspondiente y el ticket se marca como canjeado.
- Los reportes de ventas se actualizan con el nuevo tipo de documento y se preserva la trazabilidad del canje.
