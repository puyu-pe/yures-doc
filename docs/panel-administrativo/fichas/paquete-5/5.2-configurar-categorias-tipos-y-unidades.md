# 5.2 Cómo configurar categorías, tipos y unidades en Panel Administrativo

## Ruta
Catálogo y Reglas Operativas → Categorías / Tipos / Unidades

## Objetivo
Definir clasificaciones usadas en operación y control (sin teoría).

## Precondiciones
- Haber identificado las categorías de productos, tipos de receta y unidades de medida que se requieren.

## Pasos
1. Ingresar a **Categorías**, **Tipos** o **Unidades** dentro de **Catálogo y Reglas Operativas** según la clasificación a configurar.
2. Para crear un nuevo registro, hacer clic en **Agregar**; escribir el nombre y descripción de la categoría, tipo o unidad.
3. Guardar los cambios y repetir el proceso para todas las clasificaciones necesarias.
4. Revisar la lista para confirmar que las nuevas entradas aparezcan correctamente y estén disponibles al crear ítems de carta.

## Resultado esperado
- Las categorías, tipos y unidades quedan registradas en el sistema para ser utilizadas al definir productos.
- Se garantiza consistencia en el catálogo y en los reportes operativos.
