# 5.5 Cómo configurar presentaciones (equivalencias) en Panel Administrativo

## Ruta
Catálogo y Reglas Operativas → Presentaciones

## Objetivo
Configurar equivalencias para mejorar inventario y recetas, evitando errores de unidades.

## Precondiciones
- Conocer las equivalencias de unidades entre presentación comercial y consumo interno (por ejemplo, caja vs unidad).

## Pasos
1. Ingresar al módulo **Presentaciones** en **Catálogo y Reglas Operativas**.
2. Crear una nueva presentación indicando el producto, unidad de compra, unidad de consumo y factor de conversión.
3. Guardar la presentación y revisar que esté disponible al registrar inventario o recetas.

## Resultado esperado
- Las equivalencias quedan registradas y el sistema realiza conversiones automáticas entre unidades.
- Se disminuyen errores en la gestión de inventarios y el costeo de recetas.
