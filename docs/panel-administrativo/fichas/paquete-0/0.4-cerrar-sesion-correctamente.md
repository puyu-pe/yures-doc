# 0.4 Cómo cerrar sesión correctamente en Panel Administrativo

## Ruta
Menú de usuario → Cerrar sesión

## Objetivo
Cerrar sesión por seguridad y auditoría, especialmente en PCs de caja o administración.

## Precondiciones
- Estar logueado en el sistema.

## Pasos
1. Localizar el nombre del usuario o la opción **Salir** en el menú lateral inferior o en la esquina superior derecha.
2. Hacer clic en **Cerrar sesión** o **Salir** según la ubicación de la opción.
3. Confirmar la acción en el diálogo (si corresponde) para finalizar la sesión actual.
4. Esperar a que el sistema redireccione a la página de login, asegurándose de que nadie más pueda usar la cuenta.

## Resultado esperado
- El sistema muestra nuevamente la pantalla de inicio de sesión.
- La sesión del usuario se cierra y se registra en el historial para auditoría.
