# 0.0 Cómo ingresar al login del panel administrativo desde el escritorio

## Ruta
Escritorio de Windows → Acceso directo **Sistema Panel Administrativo** → Pantalla principal de Panel Administrativo → Botón **Supervisor** → Login Panel Administrativo Admin.

## Objetivo
Abrir correctamente el acceso al panel administrativo desde el escritorio hasta visualizar la interfaz de inicio de sesión.

## Precondiciones
- Tener una PC encendida con acceso al escritorio de Windows.
- Contar con el acceso directo **Sistema Panel Administrativo** visible en el escritorio.
- Disponer de conexión a internet.

## Pasos
1. En el escritorio de Windows, ubica y haz doble clic en el acceso directo **Sistema Panel Administrativo**.
2. Espera a que se abra la pantalla principal de Panel Administrativo con las opciones **Punto de venta** y **Supervisor**.
3. Haz clic en **Supervisor**.
4. Si aparece el mensaje de confirmación para acceder al supervisor, haz clic en **Sí**.
5. Espera la apertura del navegador hasta visualizar la página **Login Panel Administrativo Admin**.
6. Verifica que la barra de direcciones muestre la ruta del panel administrativo.

## Resultado esperado
- Se muestra la pantalla de inicio de sesión de **Panel Administrativo Admin** con los campos de usuario y contraseña.
- Queda listo el acceso para continuar con la ficha **0.1 Cómo iniciar sesión en Panel Administrativo**.
