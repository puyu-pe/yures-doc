# 0.2 Cómo ubicarse en el panel administrativo en Panel Administrativo

## Ruta
Panel → Resumen gerencial

## Objetivo
Identificar navegación general: menú lateral, breadcrumb/ruta, filtros frecuentes, paginación y acciones típicas (Buscar/Imprimir/Exportar).

## Precondiciones
- Haber iniciado sesión en el sistema.

## Pasos
1. Desde el panel principal observar el menú lateral y desplazar el cursor para identificar las distintas secciones (Ventas, Caja, Finanzas, etc.).
2. Revisar la ruta de navegación o **breadcrumb** en la parte superior para saber en qué módulo se encuentra.
3. Seleccionar cualquier módulo y ubicar los filtros disponibles (origen, fechas, estado) y los controles de paginación en la parte inferior de las tablas.
4. Probar las acciones frecuentes haciendo clic en **Buscar**, **Imprimir** o **Exportar** para familiarizarse con sus resultados.

## Resultado esperado
- El usuario reconoce la estructura del panel y las rutas de navegación.
- Se visualizan resultados filtrados y opciones de impresión o exportación según el módulo seleccionado.
