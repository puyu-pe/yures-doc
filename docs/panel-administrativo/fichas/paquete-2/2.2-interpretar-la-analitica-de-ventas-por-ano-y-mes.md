# 2.2 Cómo interpretar la analítica de ventas por año y mes en Panel Administrativo

## Ruta
Ventas y Documentos → Analítica de ventas

## Objetivo
Leer picos, tendencias y comparativos mensuales.

## Precondiciones
- Contar con ventas registradas para el periodo de análisis.

## Pasos
1. Ingresar a **Analítica de ventas** en el módulo **Ventas y Documentos**.
2. Seleccionar el año de interés para que el sistema muestre los gráficos mensuales de ventas.
3. Revisar las barras o líneas que indican el total vendido por mes, identificando picos y caídas.
4. Comparar la tendencia con el mismo periodo del año anterior si el gráfico lo permite.

## Resultado esperado
- Se visualiza un gráfico con las ventas de cada mes del año seleccionado.
- El usuario identifica meses con mayor o menor venta para tomar decisiones.
