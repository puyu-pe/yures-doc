# 2.3 Cómo analizar ventas por categoría o producto en Panel Administrativo

## Ruta
Ventas y Documentos → Analítica de ventas (segmentación)

## Objetivo
Identificar qué se vende y qué no para tomar decisiones de carta y precios.

## Precondiciones
- Las categorías y productos deben estar correctamente configurados en el catálogo.

## Pasos
1. Acceder al módulo **Analítica de ventas** y seleccionar la opción de segmentación por categoría o producto.
2. Elegir el rango de fechas o periodo a analizar.
3. Visualizar el gráfico o tabla que presenta la participación de cada categoría o producto en las ventas totales.
4. Identificar los ítems con mayores y menores ventas para evaluar ajustes en la carta o promociones.

## Resultado esperado
- El sistema muestra la distribución de ventas por categoría o producto en el periodo seleccionado.
- Se reconocen platos con alta y baja rotación para estrategias comerciales.
