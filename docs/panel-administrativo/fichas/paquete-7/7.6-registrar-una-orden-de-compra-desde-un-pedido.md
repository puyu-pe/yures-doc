# 7.6 Cómo registrar una orden de compra desde un pedido en Panel Administrativo

## Ruta
Compras y Proveedores → Órdenes de compra

## Objetivo
Generar orden relacionada a un pedido y validar proveedor/condiciones.

## Precondiciones
- Disponer de un pedido de compra aprobado.

## Pasos
1. Ingresar a **Órdenes de compra** y seleccionar la opción **Crear desde pedido**.
2. Elegir el pedido de compra aprobado; el sistema cargará automáticamente los productos y cantidades.
3. Revisar la información importada, ajustar precios si es necesario y verificar el proveedor y almacén de destino.
4. Guardar la orden y dejarla lista para aprobación y emisión.

## Resultado esperado
- La orden de compra se genera con los datos del pedido seleccionado.
- Se mantiene la trazabilidad entre pedido y orden para auditoría.
