# 7.12 Cómo usar el listado de compras en YURES

## Ruta
Compras y Proveedores → Listado de compras

## Objetivo
Consultar historial por periodo y aplicar filtros.

## Precondiciones
- Haber registrado compras en el sistema.

## Pasos
1. Entrar a **Listado de compras** dentro de **Compras y Proveedores**.
2. Seleccionar el rango de fechas a consultar y, si lo desea, filtrar por proveedor o estado.
3. Hacer clic en **Buscar** para ver todas las compras registradas en el periodo.
4. Revisar la lista y, si es necesario, exportar o imprimir el reporte.

## Resultado esperado
- Se presenta el historial de compras según los filtros seleccionados.
- El usuario puede realizar análisis y revisiones de compras efectuadas.
