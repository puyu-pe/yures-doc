# 7.8 Cómo verificar la relación pedido ↔ orden de compra en Panel Administrativo

## Ruta
Compras y Proveedores → Órdenes de compra

## Objetivo
Validar consistencia pedido–orden–compra desde el documento asociado.

## Precondiciones
- Haber generado la orden a partir de un pedido de compra.

## Pasos
1. En **Órdenes de compra**, localizar la orden que se originó a partir de un pedido.
2. Abrir la orden y revisar el enlace o referencia al pedido de compra original.
3. Verificar que los productos, cantidades y precios coinciden entre el pedido y la orden.
4. Documentar cualquier diferencia para ajustar la orden o informar al área correspondiente.

## Resultado esperado
- La relación entre pedido y orden se muestra claramente en los documentos.
- Se asegura la trazabilidad entre las solicitudes de compra y las órdenes emitidas.
