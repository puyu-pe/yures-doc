# 9.2 Revisar pestaña Reporte (si se usa para control diario)

## Para qué sirve

Revisar el **reporte de límite de ítems** para controlar cuántos productos ya se vendieron y cuántos quedan disponibles durante el turno o día.

---

## Checklist rápido

* Los límites de productos ya fueron configurados.
* Se necesita verificar disponibilidad restante.
* Se desea revisar control de ventas del día.

---

## Pasos

1. En la pantalla principal del POS, presionar **LIMITE** en la barra inferior.
2. Se abrirá la ventana **Registro límite de ítems**.
3. Presionar **Reporte**.
4. Revisar la tabla que muestra:

   * **Item**
   * **Total** (cantidad definida para el límite)
   * **Saldo** (cantidad restante disponible).
5. Si se requiere, presionar **Imprimir reporte** para generar el documento.

---

## Resultado esperado

* Se visualiza el listado de productos con límite configurado.
* Se puede identificar cuántos productos **quedan disponibles**.
* Permite controlar ventas y evitar vender productos agotados.

---

## Errores comunes

**No aparece ningún producto en el reporte**
→ No se configuraron límites de ítems previamente.

**El saldo llega a 0 y aún intentan vender**
→ El sistema puede bloquear la venta según configuración.

**El reporte no coincide con lo esperado**
→ Revisar si el límite fue configurado correctamente o si el turno cambió.
