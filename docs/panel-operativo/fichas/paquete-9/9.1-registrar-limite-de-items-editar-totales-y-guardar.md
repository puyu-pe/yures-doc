# 9.1 Registrar límite de ítems: editar totales y Guardar

## Para qué sirve

Definir cuántas unidades de un producto se pueden vender durante el turno o día, evitando vender más de lo disponible.

---

## Checklist rápido

* El producto tiene control de límite configurado.
* Conocer el stock disponible para venta.
* Verificar que el límite corresponda al día o turno actual.

---

## Pasos

1. Presionar el botón **LÍMITE** en la barra inferior del POS.
2. Se abrirá la ventana **Registro límite de ítems**.
3. En la pestaña **Modificar**, ubicar el producto.
4. Editar el número en la columna **TOTAL**.
5. Presionar **GUARDAR** para aplicar el cambio.
6. Cerrar la ventana.

---

## Resultado esperado

* El sistema registra el límite de venta para ese ítem.
* Cuando el producto alcance ese total, ya no podrá venderse desde el POS.

---

## Errores comunes

**No se guardó el cambio**
→ Se editó el número pero no se presionó **Guardar**.

**El límite no se respeta**
→ El producto puede no tener control de límite configurado.

**Se vendió más de lo disponible**
→ El límite se configuró después de haber realizado ventas.

**Producto desaparece del POS**
→ El límite llegó a **0** o al máximo configurado.
