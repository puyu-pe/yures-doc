# 6.3 Venta CALLE (mostrador/rápido): pedido directo → liquidación

## Para qué sirve

Registrar y cobrar una venta rápida en mostrador, sin usar mesa.

---

## Checklist rápido

* Canal **CALLE** seleccionado.
* Productos agregados al pedido.
* Pedido **enviado**.
* Datos de cliente registrados si el sistema lo solicita.

---

## Pasos

1. Presionar **CALLE** en el panel lateral.
2. Seleccionar la **categoría de producto**.
3. Agregar el producto al pedido.
4. Verificar el ítem en la lista de pedido.
5. Presionar **Enviar** para registrar el pedido.
6. Ingresar **datos del cliente** si el sistema lo solicita.
7. Confirmar con **Aceptar**.
8. Continuar con **liquidación y cobro**.

---

## Resultado esperado

* El pedido queda registrado.
* Se procede a la pantalla de cobro.
* Se puede emitir el comprobante correspondiente.

---

## Errores comunes

**No aparece el pedido para cobrar**
→ No se presionó **Enviar** después de agregar los productos.

**El sistema pide datos del cliente**
→ Debe completar nombre o documento para continuar.

**El producto no aparece en el pedido**
→ No se seleccionó correctamente desde la categoría.

**El pedido no se registra**
→ Verificar que se presionó **Enviar** antes de cerrar.
