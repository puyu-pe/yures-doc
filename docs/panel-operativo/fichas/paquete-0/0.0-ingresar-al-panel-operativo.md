# 0.0 Cómo ingresar al panel operativo desde el escritorio

## Objetivo
Acceder al **Panel operativo** para iniciar la atención en punto de venta.

## Precondiciones
- Contar con usuario y contraseña habilitados para operación.
- Tener conexión a internet en el equipo de trabajo.

## Pasos
1. Abrir el navegador web desde el escritorio.
2. Ingresar a la URL del panel operativo: <https://yures.puyu.pe/pos/>.
3. Verificar que se cargue la pantalla de inicio de sesión del entorno operativo.
4. Confirmar que se muestran campos para usuario y contraseña.

## Resultado visible esperado
- Se visualiza el acceso al panel operativo listo para iniciar sesión.
- El usuario queda preparado para continuar con la ficha **0.1 Iniciar sesión** cuando esté disponible.
