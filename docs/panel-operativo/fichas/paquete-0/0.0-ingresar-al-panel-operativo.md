# 0.0 Cómo ingresar al panel operativo desde el escritorio

## Objetivo
Acceder al **Panel operativo** para iniciar la atención en punto de venta.

## Precondiciones
- Contar con usuario y contraseña habilitados para operación.
- Tener conexión a internet en el equipo de trabajo.
- Tener visible en el escritorio el ícono **Sistema YURES**.

## Pasos
1. En el escritorio de Windows, ubicar el ícono **Sistema YURES**.
2. Hacer doble clic en **Sistema YURES** para abrir YURES.
3. Esperar a que cargue la pantalla principal del sistema.
4. Verificar que ya se visualiza el **Panel operativo**.

## Resultado visible esperado
- Se muestra el **Panel operativo** luego de abrir **Sistema YURES**.
- El usuario queda preparado para continuar con la ficha **0.1 Iniciar sesión** cuando esté disponible.
