# 0.1 Ingresar al POS: diferencia entre Punto de venta y Supervisor

## Para qué sirve

Ingresar al sistema según el tipo de función que realizará el usuario: operar ventas o realizar acciones de control.

---

## Checklist rápido

* Usuario asignado correctamente.
* Conocer la clave de acceso.
* Saber si realizará **ventas** o **acciones de supervisión**.

---

## Qué significa cada opción

**Punto de venta**
→ Permite operar el sistema: tomar pedidos, gestionar mesas, cobrar ventas e imprimir comprobantes.

**Supervisor**
→ Permite realizar acciones de control o autorización, según la configuración del sistema.

---

## Pasos

1. Ingresar la **clave** usando el teclado numérico.
2. Seleccionar el tipo de acceso:

   * **Punto de venta** para operar ventas.
   * **Supervisor** para acciones de control.
3. Confirmar el ingreso.

---

## Resultado esperado

* El usuario accede al módulo correspondiente.
* El sistema registra **usuario y terminal** en la parte superior.
* Se muestra el entorno de trabajo (mesas o funciones de control).

---

## Errores comunes

**No puede realizar ciertas acciones**
→ Probablemente ingresó como **Punto de venta** y la acción requiere **Supervisor**.

**Ingresó como Supervisor pero no ve mesas**
→ Debe ingresar por **Punto de venta** para operar ventas.

**No puede acceder al sistema**
→ La clave ingresada es incorrecta.
