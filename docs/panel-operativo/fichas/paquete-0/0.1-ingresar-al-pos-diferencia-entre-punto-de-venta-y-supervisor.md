# 0.1 Ingresar al POS: diferencia entre Punto de venta y Supervisor

## Para qué sirve

Ingresar al sistema según el tipo de función que va a realizar el usuario: operar ventas o supervisar/controlar.

---

## Checklist rápido

* Usuario asignado correctamente.
* Clave conocida.
* Saber si realizará ventas o acciones de control.

---

## Qué significa cada opción

**Punto de venta**
→ Uso operativo diario: tomar pedidos, cobrar, imprimir, trabajar en mesas o calle.

**Supervisor**
→ Acciones de control: autorizaciones, anulaciones, correcciones y revisión de operación.

---

## Pasos

1. Ingresar usuario (según configuración).
2. Digitar clave usando el teclado numérico.
3. Elegir:

   * **Punto de venta** si realizará ventas.
   * **Supervisor** si realizará acciones de control.
4. Confirmar ingreso.

---

## Resultado esperado

* Acceso correcto al módulo correspondiente.
* Se muestra el entorno según el rol elegido.
* Se registra el usuario y terminal en la parte superior.

---

## Errores comunes

**Ingresó como Supervisor para vender**
→ Puede no ver el entorno normal de mesas o flujo de venta.

**Ingresó como Punto de venta y no puede anular**
→ Esa acción puede requerir Supervisor.

**Clave incorrecta**
→ No permite acceso.
