# 0.4 Botones base (barra inferior): CAJA / BÚSQUEDA VENTA / EGRESOS-INGRESOS / LÍMITE

## Para qué sirve

Acceder rápidamente a funciones operativas clave sin salir de la pantalla principal de mesas.

---

## Checklist rápido

* Estar dentro del **Punto de venta**.
* Identificar la **barra inferior** del sistema.
* Usar el botón correcto según la acción que se desea realizar.

---

## Qué hace cada botón

**CAJA**
→ Permite acceder a funciones relacionadas con la caja del turno.

**BÚSQUEDA VENTA**
→ Permite localizar una venta o comprobante para revisarlo, imprimirlo o realizar acciones sobre él.

**EGRESOS-INGRESOS**
→ Permite registrar movimientos de dinero que no provienen de ventas (ejemplo: gastos o ingresos administrativos).

**LÍMITE**
→ Permite definir o ajustar el límite de disponibilidad de productos para evitar vender más de lo permitido.

---

## Cómo usarlo

1. Ubicar la **barra inferior** del sistema.
2. Seleccionar el botón según la acción requerida:

   * **CAJA**
   * **BÚSQUEDA VENTA**
   * **EGRESOS-INGRESOS**
   * **LÍMITE**
3. Realizar la acción dentro del módulo correspondiente.

---

## Resultado esperado

* Se abre el módulo correspondiente al botón seleccionado.
* El usuario puede ejecutar la acción sin salir del punto de venta.

---

## Errores comunes

**No encuentra una venta reciente**
→ Usar **BÚSQUEDA VENTA** en lugar de revisar mesas.

**Se registra un gasto como venta**
→ Debe registrarse en **EGRESOS-INGRESOS**.

**Producto sigue vendiéndose sin stock**
→ Verificar límites en el botón **LÍMITE**.

**Se intenta cerrar caja desde otra pantalla**
→ Ingresar primero al botón **CAJA**.
