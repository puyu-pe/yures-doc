# 0.3 Canales rápidos: CALLE y DELIVERY (qué cambia y cuándo usarlo)

## Para qué sirve

Acceder rápidamente a ventas que **no usan mesas**, como pedidos en mostrador o pedidos de reparto.

---

## Checklist rápido

* ¿El cliente está en el local y paga directo? → **CALLE**
* ¿El pedido será entregado a domicilio? → **DELIVERY**
* Confirmar canal correcto antes de registrar el pedido.

---

## Qué cambia entre los canales

**CALLE**
→ Venta rápida en mostrador o para llevar.
→ No utiliza mesas.
→ El pedido se registra y normalmente se liquida de inmediato.

**DELIVERY**
→ Pedido destinado a **entrega a domicilio**.
→ Permite registrar datos del pedido y controlar su entrega.
→ Puede quedar pendiente hasta ser entregado.

---

## Pasos

1. Ubicar los botones **CALLE** y **DELIVERY** en el panel lateral derecho.
2. Presionar el canal según el tipo de pedido.
3. Registrar los productos del pedido.
4. Continuar el flujo normal de venta según el canal.

---

## Resultado esperado

* El pedido se registra bajo el canal seleccionado.
* No se ocupa ninguna mesa del salón.
* El flujo de venta se adapta al tipo de pedido (mostrador o reparto).

---

## Errores comunes

**Registrar una venta de mostrador en una mesa**
→ Ocupa una mesa innecesariamente.

**Registrar delivery como venta normal**
→ Se pierde el control del pedido de reparto.

**Elegir mal el canal**
→ Puede afectar reportes o control de pedidos.
