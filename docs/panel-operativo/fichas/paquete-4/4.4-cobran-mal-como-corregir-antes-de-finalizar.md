# 4.4 Cobran mal

## Cómo corregir antes de finalizar

### Para qué sirve

Corregir el medio de pago o el monto antes de finalizar la venta, evitando errores de caja o cobros incorrectos.

---

### Checklist rápido

* Verificar **total de la venta**.
* Confirmar **medio de pago correcto**.
* Revisar **monto recibido**.
* Validar **vuelto o saldo pendiente**.

---

### Pasos

1. Revisar el **total de la venta** en la pantalla de liquidación.
2. Verificar que el **medio de pago seleccionado** sea el correcto.
3. Si el cliente paga con más de un medio, usar **Multi-pagos**.
4. Confirmar el **monto recibido**.
5. Revisar el cálculo de **vuelto** o saldo.
6. Presionar **Registrar Pago**.
7. Presionar **Guardar** para finalizar.

---

### Resultado esperado

* La venta queda registrada con el medio de pago correcto.
* El sistema refleja el pago completo.
* No queda saldo pendiente ni descuadre.

---

### Errores comunes

**Se registró el medio de pago equivocado**
→ Corregir antes de presionar **Registrar Pago**.

**Se presionó Guardar sin registrar pago**
→ La venta queda pendiente.

**Monto incorrecto ingresado**
→ Verificar el monto antes de registrar el pago.

**El vuelto no coincide**
→ Revisar el monto recibido antes de confirmar.
