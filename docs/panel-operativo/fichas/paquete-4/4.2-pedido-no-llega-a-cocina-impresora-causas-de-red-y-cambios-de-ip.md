# 4.2 Pedido no llega a cocina (impresora)

## Causas de red y cambios de IP

Cuando el pedido está guardado pero no imprime, y la impresora está encendida con papel, el problema casi siempre es de red.

No es el POS.
Es comunicación.

---

# 1️⃣ La IP de la impresora cambió

Este es el caso más común.

### ¿Por qué ocurre?

* Se reinició el router.
* Se cortó la luz.
* La impresora está en DHCP automático.
* El router reasignó otra IP.

El sistema sigue intentando imprimir a la IP antigua.

Resultado:
No imprime.

---

## Cómo detectar en 30 segundos

1. Imprimir hoja de autodiagnóstico desde la impresora.
2. Revisar IP actual.
3. Compararla con la IP configurada en el sistema.

Si no coinciden → ahí está el problema.

---

# 2️⃣ La impresora está en otra red

Errores frecuentes:

* Conectada al WiFi del vecino.
* Conectada al WiFi de invitados.
* Cable conectado a otro router.

Si el POS está en red 192.168.1.x
y la impresora en 192.168.0.x

No se ven.

---

# 3️⃣ Router reiniciado y perdió configuración

Algunos routers baratos:

* Cambian rango de red.
* Desactivan puertos.
* Reasignan IPs dinámicas.

Si antes funcionaba y hoy no, revisar router primero.

---

# 4️⃣ Cable de red defectuoso

Puede estar:

* Medio conectado.
* Con falso contacto.
* Dañado internamente.

La impresora puede estar encendida pero sin red.

---

# 5️⃣ Conflicto de IP duplicada

Ocurre cuando:

* Se asignó IP manual que ya estaba en uso.
* Otro dispositivo tomó esa IP.

Resultado:
Intermitencia o caída total de impresión.

---

# Buenas prácticas obligatorias

## ✔ Usar IP fija en impresoras de cocina

Nunca dejar en DHCP automático.

Asignar:

* IP fija fuera del rango automático del router.
* Ejemplo:
  Router entrega 192.168.1.100–150
  Impresora usar 192.168.1.200

---

## ✔ Documentar IPs

Tener registro físico:

* Impresora cocina
* Impresora barra
* Impresora caja

Sin documentación, el soporte se vuelve improvisación.

---

## ✔ Evitar WiFi para cocina

WiFi = inestabilidad.

Preferir:
Cable LAN directo al router.

---

# Error común

Reinstalar el sistema pensando que es software.

No lo es.

El 80% de fallas de cocina son red o IP.

---

# Señal clara de problema de red

* El pedido se guarda.
* No sale impresión.
* Reimprimir tampoco funciona.
* Impresora encendida y sin error físico.

Eso es comunicación fallida.
