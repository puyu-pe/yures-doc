# 4.9 Liberar mesa: cuándo sí y cuándo no

## Para qué sirve

Liberar una mesa que quedó ocupada en el sistema por error, permitiendo volver a usarla sin afectar otras mesas.

---

## Checklist rápido

* Confirmar que **no existe consumo pendiente**.
* Verificar que **la mesa no esté juntada con otra**.
* Validar que **no haya pedido en proceso**.
* Confirmar con el usuario responsable si hubo consumo real.

---

## Cuándo usar Liberar mesa

**Mesa quedó abierta por error**
→ Se abrió una mesa accidentalmente y no se registró ningún pedido.

**Mesa quedó ocupada después de cancelar un pedido**
→ El pedido fue eliminado o anulado antes de enviarse.

**Mesa quedó bloqueada en el sistema**
→ El cliente ya se retiró y la mesa sigue apareciendo ocupada.

---

## Cuándo NO usar Liberar mesa

**La mesa tiene consumo pendiente**
→ Debe liquidarse la venta antes.

**La mesa está juntada con otra**
→ Primero separar mesas.

**El pedido ya fue enviado a cocina**
→ Debe corregirse mediante el flujo de operación correspondiente.

---

## Pasos

1. Seleccionar la mesa ocupada en el croquis.
2. Verificar que no exista pedido activo.
3. Presionar **Liberar mesa**.
4. Confirmar la acción si el sistema lo solicita.

---

## Resultado esperado

* La mesa vuelve a estado disponible.
* El croquis muestra la mesa libre.
* Se puede usar nuevamente para una nueva venta.

---

## Errores comunes

**Liberar una mesa con consumo real**
→ Se pierde control del pedido y puede generar descuadre.

**La mesa sigue ocupada**
→ Puede estar juntada con otra mesa.

**Liberar para evitar cobrar**
→ No es el flujo correcto; debe liquidarse o corregirse la venta.
