# 5.1 Entrar a Búsqueda venta (barra inferior)

## Para qué sirve

Acceder al buscador de ventas para consultar, reimprimir, anular o revisar comprobantes emitidos.

---

## Checklist rápido

* Estar dentro del **Punto de venta**.
* Saber el **número de documento** o el **intervalo de fecha** a consultar.
* Verificar que la venta ya fue registrada.

---

## Pasos

1. Ubicar la **barra inferior del sistema**.
2. Presionar el botón **BÚSQUEDA VENTA**.
3. Esperar que se abra la pantalla de búsqueda de documentos.

---

## Resultado esperado

* Se abre el módulo de **búsqueda de ventas**.
* Permite localizar comprobantes emitidos desde el POS.
* Se pueden realizar acciones como **visualizar, reimprimir o revisar la venta**.

---

## Errores comunes

**No encuentra el botón**
→ Está en la **barra inferior**, junto a **CAJA** y **LÍMITE**.

**No aparece la venta buscada**
→ Verificar fecha o número de documento.

**Intentar buscar antes de registrar la venta**
→ Solo aparecen ventas ya guardadas en el sistema.
