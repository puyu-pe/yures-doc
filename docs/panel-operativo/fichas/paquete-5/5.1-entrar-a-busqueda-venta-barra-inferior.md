# 5.1 Entrar a **Búsqueda venta** (barra inferior)

## 🎯 Objetivo

Acceder al listado de ventas registradas para consultarlas, reimprimirlas o auditarlas.

---

## 📍 Dónde está

En la barra inferior del sistema.

Botón azul: **BÚSQUEDA VENTA**

Está al lado de:

* CAJA
* LÍMITE
* EGRESOS/INGRESOS

---

## ✅ Cómo ingresar

1. Desde el plano de mesas.
2. Presiona **BÚSQUEDA VENTA**.
3. Se abrirá la pantalla de listado de ventas.

No necesitas cerrar la mesa actual para entrar.

---

## 🧭 Para qué sirve

Desde aquí puedes:

* Buscar una venta por número.
* Buscar por fecha.
* Buscar por cliente.
* Ver detalle.
* Reimprimir.
* Auditar movimientos.

No es para modificar pedidos activos.
Es para consultar ventas registradas.

---

## ⚠️ Error común

Intentar buscar una mesa activa desde aquí.

Las mesas activas se revisan desde el plano.
La búsqueda es para ventas ya registradas o cerradas.

---

## Señal de que ingresaste correctamente

Debes ver:

* Filtros de búsqueda.
* Listado de ventas.
* Opciones de acción (ver, imprimir, etc.).
