# 5.3 Imprimir desde búsqueda (casos típicos)

## Para qué sirve

Reimprimir un comprobante ya emitido cuando el cliente lo solicita o cuando la impresión original falló.

---

## Cuándo usarlo

* El cliente pide **copia del ticket o boleta**.
* La impresora **no imprimió el comprobante** en el momento del cobro.
* Se necesita **verificar el contenido del documento** antes de entregarlo.

---

## Checklist rápido

* Tipo de documento correcto (Ticket / Boleta / Factura).
* Serie correcta.
* Número de comprobante correcto.

---

## Pasos

1. Presionar **BÚSQUEDA VENTA** en la barra inferior.
2. Seleccionar el **tipo de documento**.
3. Ingresar **serie y número** del comprobante.
4. Presionar el botón **Buscar**.
5. Verificar que el documento mostrado sea el correcto.
6. Presionar **IMPRIMIR**.

---

## Resultado esperado

* Se imprime nuevamente el comprobante.
* El contenido coincide con el documento original.
* No se genera una nueva venta ni se duplica el registro.

---

## Errores comunes

**No encuentra el documento**
→ Serie o número ingresado incorrectamente.

**Imprime otro comprobante**
→ No se verificó el número antes de imprimir.

**No imprime**
→ Verificar impresora asignada o que **PUKA-HTTP** esté activo.

**Cliente pide factura pero aparece ticket**
→ El documento original fue emitido como ticket; debe realizarse un **canje** si corresponde.
