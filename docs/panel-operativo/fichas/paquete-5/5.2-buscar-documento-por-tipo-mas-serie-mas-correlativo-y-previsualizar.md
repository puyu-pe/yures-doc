# 5.2 Buscar documento por tipo + serie + correlativo y previsualizar

## Para qué sirve

Localizar rápidamente un comprobante emitido para revisarlo, imprimirlo nuevamente, canjearlo o anularlo.

---

## Checklist rápido

* Conocer el **tipo de documento**.
* Tener **serie** del documento.
* Tener **número correlativo**.

---

## Pasos

1. Presionar **BÚSQUEDA VENTA** en la barra inferior.
2. Seleccionar el **tipo de documento**:

   * Ticket
   * Boleta electrónica
   * Factura electrónica
3. Ingresar la **serie** del documento (ej. 001).
4. Ingresar el **número correlativo** (ej. 138975).
5. Presionar el botón **Buscar** (ícono de lupa).
6. Revisar la **previsualización del comprobante** en pantalla.

---

## Resultado esperado

* El sistema muestra el comprobante encontrado.
* Se visualiza el detalle completo de la venta.
* Se habilitan opciones como:

  * **Imprimir**
  * **Canjear**
  * **Anular**

---

## Errores comunes

**No encuentra el documento**
→ Verificar que el tipo de documento sea correcto.

**Serie incorrecta**
→ Confirmar la serie del comprobante (ej. 001, 002, etc.).

**Número correlativo mal digitado**
→ Revisar el número exacto del comprobante.

**Documento equivocado**
→ Confirmar antes de usar **Anular** o **Canjear**.
