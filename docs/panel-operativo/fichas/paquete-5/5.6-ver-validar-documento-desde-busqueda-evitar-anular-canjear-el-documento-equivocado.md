# 5.6 Ver / validar documento desde búsqueda: evitar anular o canjear el documento equivocado

## Para qué sirve

Revisar un documento antes de realizar acciones como **anular, canjear o reimprimir**, evitando errores sobre la venta incorrecta.

---

## Checklist rápido

* Tipo de documento correcto (Ticket / Boleta / Factura).
* Serie y correlativo coinciden.
* Cliente correcto.
* Monto total verificado.

---

## Pasos

1. Ingresar a **Búsqueda venta** desde la barra inferior.
2. Seleccionar **tipo de documento** si aplica.
3. Ingresar **serie y correlativo** o usar filtros disponibles.
4. Presionar **Buscar**.
5. Abrir el **detalle del documento**.
6. Revisar:

   * cliente
   * productos
   * montos
   * fecha y hora
7. Confirmar que es el documento correcto antes de usar:

   * **Imprimir**
   * **Canjear**
   * **Anular**

---

## Resultado esperado

* Se visualiza el documento correcto.
* Se evitan acciones sobre ventas equivocadas.
* La operación posterior (anular o canjear) se realiza sobre el documento validado.

---

## Errores comunes

**Anular el documento equivocado**
→ No se verificó serie o correlativo antes de anular.

**Canjear un ticket incorrecto**
→ No se revisó cliente o monto antes de realizar el canje.

**No encuentra el documento**
→ Revisar filtros de búsqueda o tipo de documento seleccionado.

**Documento parece correcto pero no coincide el cliente**
→ Revisar fecha, usuario o monto para confirmar la venta correcta.
