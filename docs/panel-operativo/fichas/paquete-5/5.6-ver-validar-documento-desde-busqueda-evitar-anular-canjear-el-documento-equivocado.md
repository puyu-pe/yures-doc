# 5.6 Ver / validar documento desde búsqueda: evitar anular o canjear el documento equivocado

## 🎯 Objetivo

Confirmar al 100% que el documento que estás viendo es el correcto
antes de presionar **ANULAR** o **CANJEAR**.

Un error aquí genera:

* Anulación incorrecta.
* Reemisión duplicada.
* Problemas de caja.
* Problemas tributarios.

---

# 📍 Paso 1: Buscar correctamente

Usar:

* Tipo de documento (Ticket / Boleta / Factura).
* Serie.
* Correlativo exacto.

⚠ Error frecuente: buscar solo por correlativo sin validar el tipo.

F001-5543 no es lo mismo que B001-5543.

---

# 📍 Paso 2: Validar datos clave en la vista previa

Antes de hacer cualquier acción, revisar:

### 1️⃣ Tipo de documento

Confirmar que sea:

* Ticket (si vas a canjear).
* Boleta o Factura (si corresponde anulación electrónica).

Nunca asumas por memoria.

---

### 2️⃣ Fecha y hora

Verifica:

* Fecha de emisión.
* Hora.

Si no es del día, no intentes anular.

---

### 3️⃣ Cliente

Confirmar:

* DNI o RUC.
* Nombre o razón social.

Muchos errores ocurren cuando el cliente tiene nombre similar.

---

### 4️⃣ Ítems y total

Confirmar:

* Productos.
* Cantidades.
* Total S/.

Especialmente si hay varias ventas similares (ejemplo: varias gaseosas).

---

### 5️⃣ Método de pago

Verificar:

* Efectivo.
* Tarjeta.
* Mixto.

Anular una venta con tarjeta puede requerir validación adicional.

---

# ⚠️ Escenarios de error comunes

## Error 1: Serie equivocada

Se busca 138975 y no se valida que era Ticket 001 y no Factura F001.

Resultado: se anula documento incorrecto.

---

## Error 2: Cliente similar

Dos ventas con mismo monto.
Se anula la que no corresponde.

---

## Error 3: Hora cercana

Ventas emitidas con minutos de diferencia.
Se selecciona la incorrecta por apuro.

---

# 📌 Regla práctica operativa

Antes de presionar ANULAR o CANJEAR, confirmar mentalmente:

✔ Tipo correcto
✔ Serie correcta
✔ Correlativo correcto
✔ Fecha correcta
✔ Cliente correcto
✔ Total correcto

Si uno falla, no continuar.

---

# 🔒 Por qué esto requiere disciplina

Una anulación incorrecta:

* No se puede “deshacer”.
* Deja registro permanente.
* Afecta reportes diarios.
* Puede requerir intervención administrativa.

---

# Señal de validación completa

Si puedes responder:

“Este documento corresponde exactamente a esta venta y a este cliente”

Entonces puedes continuar.

Si tienes duda, detente.
