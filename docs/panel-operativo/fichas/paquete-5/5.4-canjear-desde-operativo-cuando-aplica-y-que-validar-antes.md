# 5.4 Canjear desde operativo: cuándo aplica y qué validar antes

## Para qué sirve

Convertir un **ticket** ya emitido en una **boleta o factura electrónica**, cuando el cliente solicita comprobante después de la venta.

---

## Cuándo usarlo

* El cliente pagó con **ticket** y luego solicita **boleta o factura**.
* El cliente necesita **factura para empresa**.
* Se emitió ticket y el cliente pide **cambio de comprobante**.

---

## Checklist rápido

* Documento original encontrado correctamente.
* El documento es **ticket**.
* El ticket **no fue anulado**.
* El cliente tiene **DNI o RUC válido** si se emitirá comprobante electrónico.

---

## Pasos

1. Presionar **Búsqueda venta**.
2. Ingresar **tipo de documento, serie y correlativo**.
3. Presionar **Buscar**.
4. Verificar que el documento mostrado sea correcto.
5. Presionar **Canjear**.
6. Seleccionar tipo de documento:

   * **Boleta**
   * **Factura**
7. Ingresar **DNI o RUC** del cliente.
8. Validar nombre o razón social.
9. Presionar **Canjear** para generar el nuevo documento.

---

## Resultado esperado

* Se genera la **boleta o factura electrónica**.
* El sistema registra el **nuevo documento con correlativo automático**.
* El ticket original queda asociado al nuevo comprobante.

---

## Qué validar antes de canjear

* Que el **monto del ticket sea correcto**.
* Que el **cliente proporcione el documento correcto** (DNI o RUC).
* Que el documento buscado sea **el ticket correcto**.

---

## Errores comunes

**No encuentra el documento**
→ Verificar tipo de documento, serie y correlativo.

**El cliente da un RUC incorrecto**
→ El sistema no validará los datos correctamente.

**Se intenta canjear un documento equivocado**
→ Siempre revisar la vista previa del ticket antes de presionar **Canjear**.

**El cliente pide factura después de emitir boleta**
→ En ese caso ya no aplica canje desde operativo; debe corregirse desde el panel administrativo.
