# 2.3 Cobro Tarjeta / Yape / Plin / Transferencia / Depósito

## Elegir el medio correcto

Aquí el error no es técnico.
Es contable.

Si eliges mal el medio de pago, descuadras caja.

---

## Paso 1: Abrir selector de medio de pago

Presionar el botón que muestra el medio actual (por defecto **EFECTIVO**).

Se despliega:

* EFECTIVO
* TRANSFERENCIA
* DEPÓSITO
* YAPE BCP
* PLIN IBK
* TARJETA

---

## Paso 2: Elegir el medio real utilizado

Seleccionar exactamente cómo pagó el cliente.

Ejemplos:

* Pagó con POS → **TARJETA**
* Pagó con Yape → **YAPE BCP**
* Pagó con Plin → **PLIN IBK**
* Transferencia bancaria → **TRANSFERENCIA**
* Voucher de depósito → **DEPÓSITO**

No uses “Efectivo” si no hubo efectivo físico.

---

## Paso 3: Ingresar el monto en PAGA

En pagos electrónicos normalmente:

* El cliente paga el total exacto.
* No hay vuelto.

Ejemplo:

* Total: 137.00
* PAGA: 137.00
* VUELTO: 0.00

---

## Paso 4: Registrar Pago

Presionar **Registrar Pago**.

Esto aplica el pago al medio seleccionado.

---

## Paso 5: Guardar

Presionar **Guardar** para cerrar definitivamente la venta.

Sin Guardar, no se consolida.

---

# Error grave común

Registrar como EFECTIVO un pago que fue Yape o Tarjeta.

Consecuencia:

* Caja mostrará más efectivo del real.
* Faltará dinero al cierre.
* El medio electrónico no cuadrará contra el reporte.

Esto no es un error menor.

---

# Señal de que está correcto

* El medio elegido coincide con cómo pagó el cliente.
* Venta pasa a estado Pagado.
* El reporte de caja clasifica correctamente el ingreso.

---

# Regla operativa

El sistema no adivina el medio.
El cajero es responsable de elegirlo bien.
