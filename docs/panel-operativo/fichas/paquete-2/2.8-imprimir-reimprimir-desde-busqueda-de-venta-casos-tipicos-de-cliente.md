# 2.8 Imprimir / Reimprimir desde Búsqueda de venta (casos típicos de cliente)

## Para qué sirve

Permite volver a imprimir un comprobante cuando el cliente lo solicita nuevamente o cuando la impresión inicial falló.

---

## Checklist rápido

* Venta ya registrada en el sistema.
* Documento correcto localizado.
* Impresora encendida y con papel.
* PUKA-HTTP activo si se usa impresión por servidor.

---

## Pasos

1. Presionar **BÚSQUEDA VENTA** en la barra inferior.
2. Buscar el documento usando:

   * tipo de documento,
   * serie,
   * correlativo.
3. Seleccionar la venta encontrada.
4. Revisar el documento en la vista previa.
5. Presionar **Imprimir** o **Reimprimir**.
6. Verificar que el comprobante salga por la impresora correspondiente.

---

## Resultado esperado

* El comprobante se vuelve a imprimir correctamente.
* La venta no se modifica ni se duplica.
* El cliente recibe una nueva copia del documento.

---

## Errores comunes

**No encuentra la venta**
→ Verificar tipo de documento, serie o rango de búsqueda.

**Impresión no sale**
→ Revisar impresora encendida, papel disponible o estado de PUKA-HTTP.

**Se intenta reimprimir desde la mesa**
→ La reimpresión se realiza desde **Búsqueda de venta**, no desde el pedido.

**Se cree que se generó una nueva venta**
→ Reimprimir solo genera una copia del comprobante existente.
