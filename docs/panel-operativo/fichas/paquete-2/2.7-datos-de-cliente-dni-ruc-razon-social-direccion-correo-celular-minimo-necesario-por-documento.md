# 2.7 Datos de cliente

DNI / RUC, Razón Social, Dirección, Correo, Celular
Mínimo necesario por tipo de documento

## Para qué sirve

Registrar o validar los datos del cliente al emitir un comprobante durante la liquidación de la venta.

El sistema puede **autocompletar algunos datos** según el tipo de documento ingresado.

---

## Checklist rápido

* Tipo de documento seleccionado.
* DNI o RUC correcto.
* Verificar datos autocompletados por el sistema.

---

## Reglas según tipo de documento

**Ticket**
→ Los datos del cliente **son opcionales**.

**Boleta electrónica**

* Ingresar **DNI**.
* El sistema **completa automáticamente el nombre del cliente**.
* La **dirección no se completa automáticamente**.

**Factura electrónica**

* Ingresar **RUC**.
* El sistema **completa automáticamente la Razón Social y la Dirección**.

**Datos opcionales en cualquier documento**

* Correo electrónico
* Celular / WhatsApp

---

## Pasos

1. En **Liquidar venta**, seleccionar el tipo de documento:

   * **Ticket**
   * **Boleta electrónica**
   * **Factura electrónica**

2. Ingresar el **N° de documento**:

   * DNI para boleta.
   * RUC para factura.

3. Verificar los datos que completa el sistema:

   * **Nombre** cuando se usa DNI.
   * **Razón Social y Dirección** cuando se usa RUC.

4. Opcionalmente ingresar:

   * **Correo electrónico**
   * **Celular / WhatsApp**

5. Continuar con **Registrar Pago** y luego **Guardar**.

---

## Resultado esperado

* El comprobante se emite con los datos del cliente según el tipo de documento.
* Los datos quedan registrados en la venta.
* El documento puede imprimirse o enviarse al cliente.

---

## Errores comunes

**Factura con DNI**
→ La factura requiere **RUC válido**.

**DNI o RUC incorrecto**
→ El sistema no podrá completar los datos automáticamente.

**No verificar los datos autocompletados**
→ Puede generarse un comprobante con información incorrecta.

**No ingresar correo o celular**
→ El comprobante solo podrá entregarse impreso.
