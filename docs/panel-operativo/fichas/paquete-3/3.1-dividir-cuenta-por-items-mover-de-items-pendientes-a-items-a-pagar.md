# 3.1 Dividir cuenta por ítems

Mover de **Items pendientes** a **Items a pagar**

## Para qué sirve

Permitir que una mesa pague **solo algunos productos del pedido**, dejando los demás pendientes para pagar después.

Se usa cuando **varias personas pagan por separado** dentro de la misma mesa.

---

## Checklist rápido

* La venta está abierta en **Liquidar venta**.
* Los productos del pedido aparecen en **Items pendientes**.
* Se identificó qué productos pagará cada persona.

---

## Pasos

1. En la ventana **Liquidar venta**, presionar **Dividir cuenta**.

2. Se abrirá la pantalla **Dividir cuenta** con dos columnas:

   * **Items pendientes** → productos aún no asignados a pago.
   * **Items a pagar** → productos que se cobrarán ahora.

3. Seleccionar el producto en **Items pendientes**.

4. Presionar el botón **>** para mover el producto a **Items a pagar**.

5. Repetir el proceso hasta dejar en **Items a pagar** solo los productos que se van a cobrar.

6. Presionar **Pagar**.

7. El sistema volverá a la pantalla de **Liquidar venta** con el monto correspondiente a esos ítems.

---

## Resultado esperado

* Solo los productos seleccionados pasan a **Items a pagar**.
* El sistema calcula el **total parcial** de esos productos.
* Se puede cobrar esa parte del pedido sin afectar el resto.

---

## Errores comunes

**Pagó todo el pedido por error**
→ No se movieron los productos antes de presionar **Pagar**.

**El producto no se mueve**
→ No se seleccionó el ítem antes de presionar **>**.

**Los productos desaparecieron de la mesa**
→ Se movieron todos los ítems a **Items a pagar** y se pagó la cuenta completa.

**Confusión entre pendientes y a pagar**
→ *Items pendientes* son los que **aún no se cobran**.
→ *Items a pagar* son los que **se cobrarán ahora**.
