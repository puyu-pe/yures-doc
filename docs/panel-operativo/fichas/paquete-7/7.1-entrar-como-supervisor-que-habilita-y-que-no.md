# 7.1 Entrar como Supervisor: qué habilita y qué no

## Para qué sirve

Ingresar al sistema con permisos de **Supervisor** para realizar controles operativos y ajustes administrativos en la operación del POS.

---

## Checklist rápido

* Usuario con acceso de **Supervisor**.
* Clave correcta.
* Necesidad de realizar control o ajuste operativo.

---

## Qué habilita el modo Supervisor

Permite realizar acciones de control que no están disponibles en el uso normal del POS, por ejemplo:

* **Abrir caja** incluso si ya fue cerrada.
* Registrar **ingresos o egresos adicionales** después del cierre de caja.
* Consultar el **reporte de supervisor** para revisar la operación.
* Acceder al POS con permisos ampliados para control del sistema.

---

## Pasos

1. En la pantalla de ingreso, seleccionar **Supervisor**.
2. Ingresar la **clave** usando el teclado numérico.
3. Confirmar el acceso.
4. El sistema mostrará el entorno de operación con el usuario identificado como **Supervisor**.

---

## Resultado esperado

* El sistema registra el ingreso del **usuario supervisor**.
* Se habilitan funciones de control de caja y revisión operativa.
* El nombre del usuario aparece en la parte superior del sistema.

---

## Errores comunes

**Intentar entrar como Supervisor sin permisos**
→ El sistema no permitirá el acceso.

**Clave incorrecta**
→ No se podrá ingresar al sistema.

**Intentar registrar ingresos o egresos después del cierre sin Supervisor**
→ El sistema no permitirá el registro.

**Usar Supervisor para operación diaria**
→ Se recomienda usar **Punto de venta** para ventas normales.
