# 1.7 Componentes simples (elige solo uno): cómo elegir correctamente y evitar errores

En algunos grupos del combo solo se permite **una opción**.

Ejemplo:

* (2) Guarnición Papas → puedes elegir entre variantes, pero el sistema solo acepta la combinación válida.
* El indicador tipo “radio” (círculo) muestra cuál está seleccionado.

---

## Qué significa “elige solo uno”

* Solo una opción puede quedar marcada.
* Si eliges otra, la anterior se desmarca.
* No se pueden marcar ambas al mismo tiempo.

Esto no es un fallo.
Es una regla del combo.

---

## Cómo elegir correctamente

1. Ingresar al grupo (por ejemplo: Guarnición Papas).
2. Revisar las opciones disponibles.
3. Seleccionar la opción correcta.
4. Verificar que el círculo quede marcado.
5. Confirmar con **Guardar**.

Si el círculo no está marcado, no quedó seleccionado.

---

## Señal visual correcta

* Círculo azul relleno → opción activa.
* Círculo vacío → opción no seleccionada.

---

## Error típico 1

Cambiar de opción y salir sin guardar.

Resultado:

* No se actualiza el combo.
* Cocina recibe la selección anterior.

---

## Error típico 2

Pensar que cantidad = selección múltiple.

Si el grupo es de selección única:

* No puedes dividir entre dos tipos.
* Solo una variante es válida.

---

## Punto crítico

Estos componentes existen para:

* Evitar combinaciones inválidas.
* Mantener estructura del combo.
* Proteger el precio configurado.

---

## Regla práctica

Si es selección única:
✔ Debe quedar solo un círculo activo.
✔ Siempre presiona Guardar antes de cerrar.
