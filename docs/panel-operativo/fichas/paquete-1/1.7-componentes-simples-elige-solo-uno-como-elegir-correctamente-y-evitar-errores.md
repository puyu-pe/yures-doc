# 1.7 Componentes simples (elige solo uno): cómo elegir correctamente y evitar errores

## Para qué sirve

Seleccionar correctamente un **componente simple del combo**, donde solo se puede elegir **una alternativa** entre varias opciones.

---

## Checklist rápido

* El combo fue agregado al pedido.
* Se abrió la ventana **Opciones del combo**.
* El componente aparece en **blanco** (pendiente).
* Se debe elegir **solo una opción**.

---

## Pasos

1. Agregar el **combo** al pedido.
2. En la ventana **Opciones del combo**, seleccionar el componente pendiente (botón en blanco).
3. Se abrirá la pantalla **Detalles del combo**.
4. Revisar las alternativas disponibles.
5. Seleccionar **una sola opción** usando el botón circular.
6. Verificar que el botón quede **marcado**.
7. Presionar **Guardar**.

---

## Resultado esperado

* El componente queda **completado**.
* El botón del componente cambia a **verde**.
* El sistema permite continuar completando los demás componentes del combo.

---

## Errores comunes

**No selecciona ninguna opción**
→ El sistema no permitirá guardar el combo.

**Selecciona más de una opción**
→ No es posible; los componentes simples permiten **solo una selección**.

**El componente sigue en blanco**
→ No se presionó **Guardar** después de elegir la opción.

**Confundir componente simple con dinámico**
→ En componentes simples se usa **botón circular (radio)** y solo se permite una alternativa.
