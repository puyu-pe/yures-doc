# 1.8 Componentes dinámicos (elige varios): cómo completar la cantidad exacta para poder guardar

## Para qué sirve

Completar correctamente un componente de combo que permite **elegir varios ítems**, asegurando que la **cantidad exacta requerida** se cumpla para poder guardar.

---

## Checklist rápido

* El componente del combo fue abierto.
* Se muestra **Cantidad máxima** requerida.
* Los ítems del componente están visibles.
* La suma total coincide con la cantidad solicitada.

---

## Pasos

1. Agregar el **combo** al pedido.
2. En la ventana **Opciones del combo**, seleccionar el componente pendiente.
3. Revisar la **cantidad máxima** indicada.
4. Elegir uno o varios ítems disponibles.
5. Ajustar la **cantidad de cada ítem** usando el selector.
6. Confirmar que el **TOTAL** coincida con la cantidad requerida.
7. Presionar **Guardar**.

---

## Resultado esperado

* El componente queda marcado como **completado**.
* Se respeta la **cantidad exacta requerida**.
* El combo puede guardarse sin errores.

---

## Errores comunes

**No permite guardar el combo**
→ La suma de cantidades no coincide con la cantidad requerida.

**Solo se seleccionó un ítem cuando se necesitaban varios**
→ Ajustar cantidades hasta completar el total.

**Se excede la cantidad permitida**
→ Reducir cantidades hasta cumplir el límite.

**El componente sigue en blanco**
→ No se completó el total requerido antes de guardar.
