# 1.12 Cerrar vs Liberar mesa: diferencias y errores típicos

## Para qué sirve

Cerrar una mesa cuando el cliente ya pagó o **liberarla** cuando el pedido quedó abierto por error o el cliente se retiró sin consumir.

---

## Checklist rápido

* Verificar si la mesa tiene consumo registrado.
* Confirmar si la venta ya fue **liquidada o pagada**.
* Revisar si la mesa tiene pedidos enviados a cocina.
* Saber el motivo si se va a **liberar** la mesa.

---

## Cuándo usar cada opción

**Cerrar mesa**
→ Cuando el consumo ya fue pagado y la venta quedó registrada.

**Liberar mesa**
→ Cuando la mesa quedó abierta por error o el cliente se retiró sin continuar el consumo.

---

## Pasos

### Liberar mesa

1. Seleccionar la mesa ocupada.
2. Ir a **Liquidar venta**.
3. Presionar **Movimientos**.
4. Escribir la **Razón** (ejemplo: “El cliente se retiró del local”).
5. Presionar **Liberar**.

---

### Cerrar mesa

1. Seleccionar la mesa ocupada.
2. Abrir **Liquidar venta**.
3. Registrar el pago correspondiente.
4. Presionar **Guardar**.
5. La mesa queda disponible automáticamente.

---

## Resultado esperado

* La mesa vuelve a estar **disponible** en el croquis.
* El color de la mesa cambia nuevamente a **verde**.
* El tiempo de ocupación se reinicia para un nuevo cliente.

---

## Errores comunes

**Liberar una mesa con consumo real**
→ Se pierde el control del pedido y puede generar descuadre en ventas.

**Intentar cerrar sin registrar pago**
→ La venta queda pendiente.

**Mesa sigue ocupada después de pagar**
→ No se completó el proceso de liquidación.

**No registrar la razón al liberar**
→ Dificulta auditoría o revisión posterior de la operación.
