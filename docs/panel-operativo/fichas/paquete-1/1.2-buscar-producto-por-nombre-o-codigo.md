# 1.2 Buscar producto por nombre o código

## Para qué sirve

Encontrar rápidamente un producto cuando no se ubica fácilmente en las categorías del menú.

---

## Checklist rápido

* Pedido o mesa abierta.
* Cursor activo en el campo de búsqueda.
* Nombre o código del producto conocido.

---

## Pasos

1. Ubicar el campo **Buscar por nombre o código** en la parte superior del panel.
2. Escribir el **nombre del producto** o su **código**.
3. Presionar el botón **🔍 (lupa)** para realizar la búsqueda.
4. Seleccionar el producto mostrado en los resultados.

---

## Resultado esperado

* El sistema muestra los productos que coinciden con la búsqueda.
* Al seleccionar uno, el producto se agrega al pedido actual.

---

## Errores comunes

**No aparecen resultados**
→ Verifique que el nombre esté escrito correctamente o intente con otra palabra del producto.

**La lista no cambia**
→ Presione el botón **lupa** para ejecutar la búsqueda.

**La búsqueda sigue filtrada**
→ Presione el botón **X** para limpiar la búsqueda.

**Producto no aparece en resultados**
→ Puede estar deshabilitado o no disponible en esa terminal.
