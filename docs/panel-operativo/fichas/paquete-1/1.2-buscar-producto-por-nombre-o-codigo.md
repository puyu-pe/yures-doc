# 1.2 Buscar producto por nombre o código

## Para qué sirve

Encontrar rápidamente un producto sin recorrer categorías.

---

## Checklist rápido

* Producto no visible en pantalla.
* Nombre o código conocido.
* Mesa correcta seleccionada.

---

## Pasos

1. En el campo superior, escribir el **nombre** o **código** del producto.
2. Presionar el botón de **lupa** para ejecutar la búsqueda.
3. Seleccionar el producto mostrado en resultados.
4. Confirmar que el ítem aparezca en la lista izquierda.
5. Para limpiar la búsqueda, presionar el botón **X**.

---

## Cómo funciona la búsqueda

* Puede buscar por:

  * Nombre completo.
  * Parte del nombre.
  * Código del producto.
* Solo se muestran productos que coinciden con el texto ingresado.
* Al seleccionarlo, se agrega automáticamente al pedido.

---

## Resultado esperado

* Producto localizado rápidamente.
* Ítem agregado correctamente al pedido.
* No es necesario cambiar de categoría manualmente.

---

## Errores comunes

**Escribir mal el nombre**
→ No aparecerán resultados.

**Olvidar presionar la lupa**
→ La búsqueda no se ejecuta.

**No limpiar la búsqueda**
→ Puede ocultar otros productos.

**Buscar producto sin seleccionar mesa**
→ Puede generar confusión sobre dónde se registra el pedido.
