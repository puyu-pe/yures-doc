## 6.7 Cómo revisar el kardex en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Inventario, Recetas y Costeo → Kardex
- **Objetivo:** Consultar historial y valorización para auditoría y lectura gerencial.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Tener movimientos de inventario generados para el producto a consultar.
- **Pasos:**
- Acceder al módulo **Kardex**.
- Seleccionar el producto y el periodo para el cual se requiere el historial de movimientos y valorizaciones.
- Revisar el listado de entradas y salidas, con fechas, cantidades, costos unitarios y saldo valorizado.
- Exportar o imprimir el kardex si es necesario para documentación contable o auditoría.
- **Resultado visible esperado:**
- El sistema presenta el historial completo de movimientos y valorización del producto seleccionado.
- Se facilita el control de inventarios y la conciliación contable.
