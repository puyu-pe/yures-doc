## 3.1 Cómo usar la trazabilidad de operación en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Caja, Turnos y Auditoría → Trazabilidad de operación
- **Objetivo:** Reconstruir pedidos por intervalo, área, mesa, usuario y terminal para detectar patrones.
- **Tipo / Duración:** Flujo operativo (120–180 s)
- **Precondiciones:**
- Contar con permisos de auditoría para acceder a la trazabilidad.
- **Pasos:**
- Acceder al módulo **Trazabilidad de operación** en **Caja, Turnos y Auditoría**.
- Seleccionar los filtros de fecha y hora, área o mesa, usuario y terminal para acotar la búsqueda.
- Hacer clic en **Buscar** para mostrar los pedidos y eventos registrados dentro del intervalo seleccionado.
- Analizar la secuencia de acciones (creación, modificación, anulación, despacho) para detectar patrones o incidencias.
- **Resultado visible esperado:**
- El sistema presenta un historial detallado de operaciones con fechas, usuarios y acciones realizadas.
- Se pueden identificar causas de errores o incidencias operativas y documentarlas para corrección.
