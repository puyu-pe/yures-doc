# 3.1 Cómo usar la trazabilidad de operación en YURES

## Ruta
Caja, Turnos y Auditoría → Trazabilidad de operación

## Objetivo
Reconstruir pedidos por intervalo, área, mesa, usuario y terminal para detectar patrones.

## Precondiciones
- Contar con permisos de auditoría para acceder a la trazabilidad.

## Pasos
1. Acceder al módulo **Trazabilidad de operación** en **Caja, Turnos y Auditoría**.
2. Seleccionar los filtros de fecha y hora, área o mesa, usuario y terminal para acotar la búsqueda.
3. Hacer clic en **Buscar** para mostrar los pedidos y eventos registrados dentro del intervalo seleccionado.
4. Analizar la secuencia de acciones (creación, modificación, anulación, despacho) para detectar patrones o incidencias.

## Resultado esperado
- El sistema presenta un historial detallado de operaciones con fechas, usuarios y acciones realizadas.
- Se pueden identificar causas de errores o incidencias operativas y documentarlas para corrección.
