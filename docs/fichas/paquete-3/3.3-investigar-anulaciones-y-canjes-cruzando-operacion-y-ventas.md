## 3.3 Cómo investigar anulaciones y canjes cruzando operación y ventas en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Caja, Turnos y Auditoría → Trazabilidad de operación / Comandas y Kitchen tracking; ruta relacionada: Ventas y Documentos → Búsqueda y gestión de comprobantes
- **Objetivo:** Aplicar el método operación → documento → corrección para explicar incidencias con evidencia.
- **Tipo / Duración:** Flujo operativo (120–180 s)
- **Precondiciones:**
- Haber detectado una incidencia de anulación o canje en un cierre o reporte.
- **Pasos:**
- Revisar la **Trazabilidad de operación** para identificar quién creó y modificó el pedido y en qué momentos.
- Consultar **Comandas y Kitchen tracking** para verificar si el pedido fue preparado o despachado antes de la anulación/canje.
- Ir a **Búsqueda y gestión de comprobantes** para localizar el documento asociado y revisar su estado y justificante.
- Registrar la evidencia encontrada (usuarios implicados, tiempos, motivos) para sustentar la explicación de la incidencia.
- **Resultado visible esperado:**
- Se reconstruye la secuencia de eventos que llevaron a la anulación o canje.
- Se documenta evidencia para respaldar la corrección o para capacitación del equipo.
