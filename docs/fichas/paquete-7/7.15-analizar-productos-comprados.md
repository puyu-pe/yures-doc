# 7.15 Cómo analizar productos comprados en YURES

## Ruta
Compras y Proveedores → Productos comprados

## Objetivo
Analizar compras por período para detectar sobrecompra/subcompra.

## Precondiciones
- Tener registros de compras con detalles de productos y cantidades.

## Pasos
1. Acceder a **Productos comprados** en **Compras y Proveedores**.
2. Seleccionar el periodo que se desea analizar y filtrar por proveedor o categoría si corresponde.
3. Revisar la lista de productos con cantidades y montos comprados.
4. Identificar productos con compras excesivas o insuficientes para ajustar pedidos futuros.

## Resultado esperado
- Se obtiene un reporte de productos comprados con sus cantidades y montos por periodo.
- El área de compras puede tomar decisiones para optimizar el abastecimiento.
