## 7.15 Cómo analizar productos comprados en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Compras y Proveedores → Productos comprados
- **Objetivo:** Analizar compras por período para detectar sobrecompra/subcompra.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Tener registros de compras con detalles de productos y cantidades.
- **Pasos:**
- Acceder a **Productos comprados** en **Compras y Proveedores**.
- Seleccionar el periodo que se desea analizar y filtrar por proveedor o categoría si corresponde.
- Revisar la lista de productos con cantidades y montos comprados.
- Identificar productos con compras excesivas o insuficientes para ajustar pedidos futuros.
- **Resultado visible esperado:**
- Se obtiene un reporte de productos comprados con sus cantidades y montos por periodo.
- El área de compras puede tomar decisiones para optimizar el abastecimiento.
