# 7.4 Cómo generar un pedido de compra en YURES

## Ruta
Compras y Proveedores → Pedidos de compra

## Objetivo
Completar y guardar el pedido de compra (flujo completo).

## Precondiciones
- Haber creado el pedido de compra inicial.

## Pasos
1. Seleccionar el pedido en estado **pendiente** desde la lista de **Pedidos de compra**.
2. Agregar los productos necesarios, especificando cantidad y precio estimado para cada uno.
3. Revisar el resumen del pedido y, si es correcto, hacer clic en **Guardar** o **Enviar**.
4. El sistema actualizará el estado del pedido a **generado** o **en revisión** según el flujo interno.

## Resultado esperado
- El pedido de compra se completa con su detalle de productos y costos estimados.
- Queda listo para ser aprobado, rechazado o transformado en orden de compra.
