# 7.2 Cómo registrar una orden de compra en YURES

## Ruta
Compras y Proveedores → Órdenes de compra

## Objetivo
Crear orden de compra y dejarla lista para seguimiento.

## Precondiciones
- Tener proveedores y productos registrados.

## Pasos
1. Ingresar a **Órdenes de compra** y seleccionar **Nueva orden**.
2. Elegir el proveedor, la fecha de emisión y el almacén de destino.
3. Agregar productos a la orden indicando cantidades y precios estimados.
4. Guardar la orden de compra y verificar que el estado quede como pendiente o en revisión para su seguimiento.

## Resultado esperado
- La orden de compra se crea con los productos y condiciones seleccionados.
- Se genera un registro para seguimiento y aprobación posterior.
