# 7.5 Cómo registrar el detalle de un pedido de compra en YURES

## Ruta
Compras y Proveedores → Pedidos de compra

## Objetivo
Agregar productos, cantidades y costos estimados con validaciones.

## Precondiciones
- Contar con un pedido de compra en estado pendiente o generado.

## Pasos
1. Abrir el pedido de compra correspondiente desde la lista.
2. Hacer clic en **Agregar producto** e ingresar el nombre del producto, unidad, cantidad y precio estimado.
3. Repetir el paso anterior para cada producto necesario.
4. Revisar el total estimado del pedido y guardar para registrar el detalle.

## Resultado esperado
- Cada producto del pedido se registra con su cantidad y costo estimado.
- El pedido queda listo para ser aprobado o convertirlo en orden de compra.
