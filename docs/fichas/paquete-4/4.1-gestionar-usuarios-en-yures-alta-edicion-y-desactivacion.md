# 4.1 Cómo gestionar usuarios en YURES (alta, edición y desactivación)

## Ruta
Usuarios y Seguridad → Usuarios

## Objetivo
Crear, editar y desactivar usuarios con buenas prácticas de roles y nombres.

## Precondiciones
- Ser administrador o tener permisos de gestión de usuarios.

## Pasos
1. Acceder al módulo **Usuarios** dentro de **Usuarios y Seguridad**.
2. Para crear un usuario, hacer clic en **Registrar**; completar los campos de nombre, correo electrónico, contraseña y seleccionar el rol (motorista, chef, cajero, etc.); luego guardar.
3. Para editar, localizar al usuario en la lista y pulsar el ícono de lápiz; modificar los datos necesarios y confirmar.
4. Para desactivar, pulsar el ícono de papelera o desactivar en la fila correspondiente y aceptar la confirmación.

## Resultado esperado
- El nuevo usuario se añade a la lista con su rol asignado y puede iniciar sesión.
- Los cambios o desactivaciones se reflejan inmediatamente en la tabla de usuarios.
