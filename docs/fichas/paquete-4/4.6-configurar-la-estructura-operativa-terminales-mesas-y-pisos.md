## 4.6 Cómo configurar la estructura operativa (terminales, mesas y pisos) en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Configuración del sistema → Terminales / Mesas / Pisos/Plantas / Pisos por terminal
- **Objetivo:** Definir estructura operativa y relación terminal–mesas–pisos.
- **Tipo / Duración:** Flujo operativo (120–180 s)
- **Precondiciones:**
- Tener claro el número de terminales, pisos y mesas que se gestionarán en el local.
- **Pasos:**
- Acceder a **Terminales** y registrar cada equipo POS asignando un nombre y ubicación (piso o área).
- Ingresar al módulo **Pisos/Plantas** para crear los distintos niveles o áreas del negocio.
- Registrar las **Mesas** en cada piso, definiendo su numeración y capacidad.
- En la opción **Pisos por terminal**, asignar qué pisos puede gestionar cada terminal para garantizar un flujo coherente de comandas y pagos.
- **Resultado visible esperado:**
- El sistema refleja la estructura operativa completa, permitiendo asignar pedidos y mesas correctamente.
- Se facilita el trabajo de los usuarios al saber qué terminal maneja cada piso o área.
