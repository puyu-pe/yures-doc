## 4.5 Cómo configurar impresión por documento en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Configuración del sistema → Impresiones por documento
- **Objetivo:** Asignar impresoras por documento y resolver errores típicos (no imprime / doble impresión).
- **Tipo / Duración:** Flujo operativo (120–180 s)
- **Precondiciones:**
- Contar con impresoras configuradas en la red y reconocidas por el sistema.
- **Pasos:**
- Dirigirse a **Impresiones por documento** dentro de **Configuración del sistema**.
- Para cada tipo de documento, asignar la impresora correspondiente (cocina, barra, cliente) mediante los campos disponibles.
- Guardar la configuración y realizar una prueba de impresión para verificar la correcta asignación.
- Ajustar parámetros adicionales como número de copias o tamaño de papel según sea necesario.
- **Resultado visible esperado:**
- Las impresoras quedan correctamente asignadas a cada documento y la impresión funciona sin incidencias.
- Se reducen errores de impresiones duplicadas o documentos no enviados a la impresora adecuada.
