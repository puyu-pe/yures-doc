# 1.7 Cómo validar un comprobante antes de corregirlo en YURES

## Ruta
Ventas y Documentos → Búsqueda y gestión de comprobantes → Seleccionar… → Ver documento / Ver comandas

## Objetivo
Confirmar monto, usuario, hora, medio de pago y respaldo operativo con comandas.

## Precondiciones
- Haber localizado el comprobante mediante búsqueda o filtrado.

## Pasos
1. En la tabla de resultados de comprobantes, identificar el registro que se desea validar.
2. Hacer clic en el botón **Ver documento** para revisar los datos generales: cliente, productos, montos y medio de pago.
3. Abrir la opción **Ver comandas** para revisar el detalle de comandas asociadas al pedido y confirmar tiempos y cantidades servidas.
4. Verificar que la información coincide con la operación registrada antes de proceder a cualquier corrección o anulación.

## Resultado esperado
- El usuario confirma que el comprobante contiene datos correctos y se respalda con las comandas correspondientes.
- Se evita realizar anulaciones o ajustes innecesarios por información incompleta o errónea.
