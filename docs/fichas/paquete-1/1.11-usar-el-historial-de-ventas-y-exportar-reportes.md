## 1.11 Cómo usar el historial de ventas y exportar reportes en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Ventas y Documentos → Historial de ventas
- **Objetivo:** Aplicar filtros clave y exportar (PDF/Excel) para control y contabilidad.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Disponer de ventas registradas en el periodo a consultar.
- **Pasos:**
- Acceder al módulo **Historial de ventas** dentro de **Ventas y Documentos**.
- Seleccionar el rango de fechas, tipo de documento o estado que se desea consultar utilizando los filtros disponibles.
- Hacer clic en **Buscar** para visualizar las ventas correspondientes al criterio seleccionado.
- Revisar el listado obtenido y utilizar la paginación si hay muchos registros.
- Exportar el reporte a PDF o Excel mediante los botones de **Exportar** según el formato deseado.
- **Resultado visible esperado:**
- Se muestra una tabla con todas las ventas filtradas y su detalle principal.
- El usuario descarga el archivo en el formato seleccionado para respaldo contable.
