## 1.4 Cómo revisar turnos de auditoría en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Caja, Turnos y Auditoría → Gestión de Turnos
- **Objetivo:** Revisar operador, horarios, terminal y eventos relevantes antes de validar un cierre.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Contar con permisos de auditoría para acceder a la gestión de turnos.
- Tener turnos registrados en el sistema.
- **Pasos:**
- Ingresar al módulo **Caja, Turnos y Auditoría** y seleccionar la opción **Gestión de Turnos**.
- Utilizar los filtros disponibles para elegir la terminal y el rango de fechas del turno que desea revisar.
- Hacer clic en **Buscar** para listar los turnos registrados.
- Revisar la información del operador, horario de inicio y cierre, terminal asociada y eventos (aperturas, cierres, incidencias) en la tabla.
- Si necesita más detalles, seleccionar el turno para acceder a la vista ampliada o historial de acciones.
- **Resultado visible esperado:**
- Se obtiene una lista de turnos filtrados con detalles completos de cada operador y terminal.
- El auditor puede validar horarios y eventos antes de aprobar el cierre de caja.
