## 8.5 Cómo usar el plan de cuentas en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Finanzas y Gastos → Plan de cuentas
- **Objetivo:** Clasificar correctamente para reportes consistentes.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Tener definidas las cuentas contables según la normativa y la estructura de la empresa.
- **Pasos:**
- Ingresar al módulo **Plan de cuentas**.
- Revisar la lista de cuentas contables existentes y su estructura jerárquica.
- Crear nuevas cuentas si es necesario, asignándoles un código y descripción de acuerdo con la clasificación.
- Asignar las cuentas apropiadas a cada transacción para asegurar la correcta clasificación de ingresos y egresos.
- **Resultado visible esperado:**
- El plan de cuentas está actualizado y organizado para uso administrativo y contable.
- Los reportes financieros reflejan correctamente los movimientos según su cuenta contable.
