## 2.3 Cómo analizar ventas por categoría o producto en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Ventas y Documentos → Analítica de ventas (segmentación)
- **Objetivo:** Identificar qué se vende y qué no para tomar decisiones de carta y precios.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Las categorías y productos deben estar correctamente configurados en el catálogo.
- **Pasos:**
- Acceder al módulo **Analítica de ventas** y seleccionar la opción de segmentación por categoría o producto.
- Elegir el rango de fechas o periodo a analizar.
- Visualizar el gráfico o tabla que presenta la participación de cada categoría o producto en las ventas totales.
- Identificar los ítems con mayores y menores ventas para evaluar ajustes en la carta o promociones.
- **Resultado visible esperado:**
- El sistema muestra la distribución de ventas por categoría o producto en el periodo seleccionado.
- Se reconocen platos con alta y baja rotación para estrategias comerciales.
