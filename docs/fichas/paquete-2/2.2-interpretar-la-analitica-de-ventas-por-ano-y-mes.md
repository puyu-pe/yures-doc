## 2.2 Cómo interpretar la analítica de ventas por año y mes en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Ventas y Documentos → Analítica de ventas
- **Objetivo:** Leer picos, tendencias y comparativos mensuales.
- **Tipo / Duración:** CRUD/Consulta (60–120 s)
- **Precondiciones:**
- Contar con ventas registradas para el periodo de análisis.
- **Pasos:**
- Ingresar a **Analítica de ventas** en el módulo **Ventas y Documentos**.
- Seleccionar el año de interés para que el sistema muestre los gráficos mensuales de ventas.
- Revisar las barras o líneas que indican el total vendido por mes, identificando picos y caídas.
- Comparar la tendencia con el mismo periodo del año anterior si el gráfico lo permite.
- **Resultado visible esperado:**
- Se visualiza un gráfico con las ventas de cada mes del año seleccionado.
- El usuario identifica meses con mayor o menor venta para tomar decisiones.
