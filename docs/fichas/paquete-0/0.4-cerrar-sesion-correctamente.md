## 0.4 Cómo cerrar sesión correctamente en YURES

- **Sistema:** YURES
- **Módulo / Ruta:** Menú de usuario → Cerrar sesión
- **Objetivo:** Cerrar sesión por seguridad y auditoría, especialmente en PCs de caja o administración.
- **Tipo / Duración:** Onboarding (≤ 180 s)
- **Precondiciones:**
- Estar logueado en el sistema.
- **Pasos:**
- Localizar el nombre del usuario o la opción **Salir** en el menú lateral inferior o en la esquina superior derecha.
- Hacer clic en **Cerrar sesión** o **Salir** según la ubicación de la opción.
- Confirmar la acción en el diálogo (si corresponde) para finalizar la sesión actual.
- Esperar a que el sistema redireccione a la página de login, asegurándose de que nadie más pueda usar la cuenta.
- **Resultado visible esperado:**
- El sistema muestra nuevamente la pantalla de inicio de sesión.
- La sesión del usuario se cierra y se registra en el historial para auditoría.
